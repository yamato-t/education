# **分岐処理**

---

「キーボードで押下した内容を表示する」「簡単な計算機を作成する」など、  
変数を利用するだけで人の要求に答えるアプリケーションを作る事ができる。  
しかし、ゲームを作る為にはまだ色々と足りていない。

例えば格闘ゲームなどで考えると、  
相手を攻撃した際に体力がまだ残っていれば対戦が継続するが、  
体力がなくなっていれば対戦を終了させないといけない。

ゲームのような複雑なアプリケーションを作る為には、  
プログラム上で条件に応じて処理を分岐させなければならない場面が多々出てくる。

今回はその <span style ="color: red;">**分岐**</span> について学んでいく。

分岐処理は**プログラム基本構造の一つ**になる。  
以前の順次処理に加え、今回の分岐処理によって、  
作れる物の幅が大きく広がるはず。

---


## **サンプルプログラム作成**

まずは利用するサンプルプログラムを作成する。

```c++
/*
 * コンピュータが決めたランダムな数値を当てる簡易ゲームを作る
 */
#include <stdio.h>
#include <stdlib.h>

int main() {
    // 最初はこれ
    int answer = 5;

    // 次はこれ
    // コンピュータの時間（を表す値）を変数に代入する
    int t = (int)time(nullptr);
    // t を 10 で割って、その余りを変数 answer に代入する
    // 10 で割った余りなので、answer には「0～9」のどれかが入っている
    int answer = t % 10;
 
    // キーボードで押した数を変数 input に代入する
    int input;
    scanf_s("%d", &input);

    // answer と input の数値が一致している時だけ「一致！」と表示する
    printf("一致！\n");

    return 0;
}
```

ただ、このコードではやりたい事が実現できていない。  
なぜなら必ず「一致！」と表示されるようになっているから。

ここで考えるべきは、下記の部分になる。

```c++
    // answer と input の数値が一致している時だけ「一致！」と表示する
    printf("一致！\n");
```

二つの変数に入っている数値を見比べて、  
一致する時だけ表示しなければならない。

一致する場合としない場合で <span style ="color: red;">**処理を分岐**</span> させる必要が出てくる。

---


## **条件**

処理を分岐させなければならない場合、  
まず考えるべきは<span style ="color: red;">**分岐の条件**</span>になる。  
どういう条件があるのかを整理する。

「一致！」と表示する為に必要なのは
「answer と input の数値が一致している」という条件。  
この条件をプログラムでコーディングしなければならない。

そんな時に利用できるのが <span style ="color: red;">**if 文**</span> という構文。

```c++
// answer と input の数値が一致している時だけ「一致！」と表示する
if(answer == input)
{
    printf("一致！\n");
}
```

着目すべきは `if(answer == input)` の部分。  
`if` というキーワードを書いた後、`()` の中に条件を入れる。  
ここでは `answer == input` が条件文になっており、  
この `==` が「一致する」という意味を持っている。

この条件をクリアしている場合のみ、  
その下の波カッコで括られた処理が実行される。  
これで一致している時のみ「一致！」と表示されるはず。  

ちなみに、`!=` と書くと「一致しない」という意味になる。  
`==` と `!=` はセットで覚えておくべき。

---

## **「それ以外」の場合**

一致した場合は表示できるようになったが、  
一致していない時は何も表示されていない。  
これでは不親切なので「間違ってる！」と表示するように変更する。

先ほどと同様に、分岐する条件を考える。  
今回は「answer と input の数値が一致していない」という事が条件になる。

同じように if 文を書けば良い、と思われるかもしれないが、  
すでに「answer と input の数値が一致している」という条件文は書かれている。  
この場合、<span style ="color: red;">**「answer と input の数値が一致している」以外**</span>  
という条件をコーディングする事が可能。

そんな時に利用できるのが <span style ="color: red;">**if else 文**</span> という構文。

```c++
// answer と input の数値が一致している時だけ「一致！」と表示する
// それ以外の時は「間違ってる！」と表示する
if(answer == input)
{
    printf("一致！\n");
}
else
{
    printf("間違ってる！\n");
}
```

if と同じ形で `else` という部分が追加されている。  
少し異なるのは、`else` の後ろに条件を表す `()` がない点。

`else` というのは「それ以外」という意味であり、  
if ですでに一つ条件を指定している為、  
<span style ="color: red;">**それ以外の場合は全て**</span>「間違ってる！」と表示すれば良く、  
ここで条件を書く必要はない。

---


## **多数の分岐**

「間違っている！」ではなく、  
入力した値の方が大きい場合や小さい場合も判定するよう、  
処理を二つだけではなく多数に分岐させるように拡張してみる。

```c++
// answer と input の数値が一致している時は「一致！」と表示する
// answer より input の数値の方が大きい時は「大きい！」と表示する
// answer より input の数値の方が小さい時は「小さい！」と表示する
```

この場合、「大きい時」「小さい時」はそれぞれ別に条件をコーディングする必要がある。  
複数の分岐を行わせる場合、`else` だけでは不十分。  
その為、`else` の後ろに新しく条件を if 文で追加する。

```c++
    // answer と input の数値が一致している時は「一致！」と表示する
    // answer より input の数値の方が大きい時は「大きい！」と表示する
    // answer より input の数値の方が小さい時は「小さい！」と表示する
    if (answer == input)
    {
        printf("一致！\n");
    }
    else if (answer < input)
    {
        printf("大きい！\n");
    }
    else if (answer > input)
    {
        printf("小さい！\n");
    }
```

数値の大小を条件とする場合は、大なり・小なりの記号を利用する。  
手元のコードを変更して試してみてほしい。

## **複数の条件**

「大きい！」「小さい！」の表示に加えて、もう少しヒントを与える。  
`answer` と `input` の数値の差が「マイナス3より大きく、プラス3より小さい」場合に  
「惜しい！」と表示できるようにする。  

```c++
// まずは answer と input の数値の差を計算する
int sub = answer - input;

// 条件を分解する
// 「マイナス3より大きく」
// sub = 5;
if(sub > -3)
{
    printf("惜しい！\n");
}

// 「プラス３より小さい」
// sub = -4;
if(sub < 3)
{
    printf("惜しい！\n");
}
```

上記のように両方の条件を独立させていると、正しく判定をとる事ができない。  
二つの if 文を組み合わせる必要がある。方法は二つある。

```c++
// 方法１
// 「マイナス3より大きく」
if(sub > -3)
{
    // 「プラス３より小さい」
    if(sub < 3)
    {
        printf("惜しい！\n");
    }
}

// 方法２
// 「マイナス3より大きく」「且つ」「プラス３より小さい」
if(sub > -3 && sub < 3)
{
    printf("惜しい！\n");
}
```

方法２では `&&` という記述が追加されている。  
これを入れる事で「両方の条件をクリアしている」という意味になる。  
ここでいう「両方の条件」とは、`sub > -3` と `sub < 3` の事。

ここまで作ったところで、実はこのプログラムは正しく動かない。  
ある条件で「一致！」と「惜しい！」が両方表示されてしまう。  
原因を見つけて修正してみてほしい。

## **演算子**

ここまでのコーディングで登場した、二つの値を比較する記号  
（`==`、`<`、`>`）は<span style ="color: red;">**比較演算子（関係演算子）**</span>と呼ばれている。  
値の大小や一致しているかどうかを確認する場合にこの演算子を利用する。  
大なり・小なりや `==` は概ね直感的に利用できるはず。

if 文ではこの演算子の結果を利用して処理の分岐を行っているが、  
ここでいう演算子の結果とは「**真 or 偽**」のどちらかになる。  
`true` or `false` と表現されるもので、こういった値を <span style ="color: red;">**真理値**</span> と呼ぶ。

真理値を受け取る為の変数型も存在し、<span style ="color: red;">**bool 型**</span>と呼ばれている。  
bool 型は `true` か `false` のどちらかを保持でき、  
数値的には `true` は `1`、`false` は `0` を表している。

if 文で比較演算子を使った時の事を振り返ると、  
「値が一緒の時」「値が大きい時」という意味で使っていたと思うが、  
コーディングを行う上では  

- **「値が一緒、という比較の結果が true の時」**  
- **「値が大きい、という比較の結果が true の時」**

という認識が正しい。

bool 型の変数を利用すると、この様に書き換える事ができる。

```c++
// answer と input の数値が一致している時は「一致！」と表示する
if (answer == input)
{
    printf("一致！\n");
}

// answer と input が一致していれば same に true が代入される
bool same = answer == input; 
if(same == true)
{
    printf("一致！\n");
}

// same がそもそも真理値(true か false)である為、こういった書き方も可能になる
if(same)
{
    printf("一致！\n");
}
```

最後に登場した `&&` は <span style ="color: red;">**論理演算子（ブール演算子）**</span> と呼ばれる。  
二つの真理値の論理演算を行い、その結果を真理値として得る事ができる。

論理演算には以下の種類がある。

- **論理積**：A and（かつ）B　→ 両方とも真なら真、それ以外は偽
- **論理和**：A or（または）B　→ 片方が真なら真、両方とも偽なら偽

`&&` は論理積であり、両方の比較結果が `true` の場合のみ `true` となる。  
複数の条件をこのように組み合わせる事ができる。

## **条件が真理値以外の場合**

下記のコードの結果はどうなるか考えてみてほしい。

```c++
if(3)
{
    printf("表示される？\n");
}
```

これは表示される。  
`true` でも `false` でもないのに処理される。

実は if 文は「true か false か」で判断しているわけではなく、  
<span style ="color: red;">**false かそれ以外か**</span> で判断している。  
`false` は数値でいうと `0` であり、  
**0 の時は処理しない、それ以外なら処理する**という動作になる。  
なので直接 `3` という数値を条件に入れた場合 `0` ではない為に処理が実行される。

## **switch case 文による条件分岐**

ここから、結果に応じてリザルトランクを表示するようにしてみる。  
具体的には以下のような表示とする。

- 一致した場合：「ランクS」
- 惜しい場合　：「ランクA」
- それ以外　　：「ランクB」

まず以下の変数を宣言する。

```c++
    // ランクを保持する変数を宣言する
    int rank = 0;
```

次に rank へ値を代入していく。  
一致した場合は `2`、惜しい場合は `1`、それ以外は `0` をそれぞれ代入する。

```c++
/*
 * コンピュータが決めたランダムな数値を当てる簡易ゲームを作る
 */
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main() {
    // コンピュータの時間（を表す値）を変数に代入する
    int t = (int)time(nullptr);
    // random を 10 で割って、その余りを変数 answer に代入する
    // 10 で割った余りなので、answer には「0～9」のどれかが入っている
    int answer = t % 10;

    printf("%d\n", answer);

    // キーボードで押した数を変数 input に代入する
    int input;
    scanf_s("%d", &input);

    // ランクを保持する変数を宣言する
    int rank = 0;

    // answer と input の数値が一致している時は「一致！」と表示する
    if (answer == input)
    {
        rank = 2;
        printf("一致！\n");
    }
    // answer より input の数値の方が大きい時は「大きい！」と表示する
    else if (answer < input)
    {
        printf("大きい！\n");
    }
    // answer より input の数値の方が小さい時は「小さい！」と表示する
    else if (answer > input)
    {
        printf("小さい！\n");
    }

    // まずは answer と input の数値の差を計算する
    int sub = answer - input;
    if (sub > -3 && sub < 3 && sub != 0)
    {
        rank = 1;
        printf("惜しい！\n");
    }

    // rank に入っている数値に合わせて表示を変える
    ?

    return 0;
}
```

ここまでやった上で、`?` の部分はどのようにコーディングすれば良いか考えてみてほしい。

```c++
    // rank に入っている数値に合わせて表示されるランクを変える
    if(rank == 2) 
    {
        printf("ランクS\n");	
    }
    else if(rank == 1) 
    {
        printf("ランクA\n");	
    }
    else
    {
        printf("ランクB\n");	
    }
```

もちろんこれも正解。  
ただ、ここでは別の方法を使ってみる。

### **switch case 文**

条件による分岐処理を実現する方法は if else だけではない。

他に <span style ="color: red;">**switch case 文**</span> という構文もある。  
では switch case 文に書き換えていく。

```c++
    // rank に入っている数値に合わせて表示を変える
    switch (rank) 
    {
    case 2:
        printf("ランクS\n");
        break;
    case 1:
        printf("ランクA\n");
        break;
    default:
        printf("ランクB\n");
        break;
    }
```

switch case 文は <span style ="color: red;">**変数の中身によって処理を替える**</span> という事を行う。

1. **switch キーワード**  
   `switch` の後ろのカッコに `rank` 変数を指定する。  
   その下の波カッコの中に、切り替える処理内容を記述する。

2. **case キーワード**  
   `case` の後ろに `2` や `1` といった値を書く。  
   これは `rank` 変数に代入されている数を表しており、  
   数に合わせた各ランク表示が直後に実行される。  
   このように書く事で、変数の中身によって処理される内容を変える事ができる。

3. **break キーワード**  
   <span style ="color: red;">**処理を終わらせる**</span> という意味を持つ。  
   変数の数と一致する case から始まり、break まで処理されたら  
   その switch case 文を抜けていく。  
   <span style ="color: red;">**分岐は case から始まり、break で終わる**</span>。

4. **default キーワード**  
   他の case で指定されている値以外の場合、という意味になる。  
   else と同じ意味であり、省略しても問題ない。

---

## **if else 文 と switch case 文の違い**

if else 文と switch case 文はどちらも同じ分岐処理を行う構文だが、  
switch case 文は if else 文と違い、完全な分岐処理とは言えない。  
switch case は <span style ="color: red;">**処理の開始場所を変える事で分岐しているだけ**</span> である為。

このコードを試してみてほしい。

```c++
    // rank に入っている数値に合わせて表示を変える
    switch (rank) 
    {
    case 2:
        printf("ランクS\n");
    case 1:
        printf("ランクA\n");
    default:
        printf("ランクB\n");
        break;
    }
```

- `rank` が `2` の場合：ランクSだけでなく、ランクAとランクBも表示される  
- `rank` が `1` の場合：ランクAとランクBが表示される

`break` を消した事で「ここまで処理する」という区切りがなくなり、  
次の `case` の処理も続けて実行されてしまっている。

if else 文の場合は一つの条件を満たして処理されると、  
他の分岐処理は実行されない。  
しかし switch case 文は処理の開始位置が変わっているだけなので、  
`break` で明確に終わらせないと次の `case` も処理されてしまう点に注意が必要。

## **纏め**

最終的なコードは以下のようになる。

```c++
/*
 * コンピュータが決めたランダムな数値を当てる簡易ゲームを作る
 */
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main() {
    // コンピュータの時間（を表す値）を変数に代入する
    int t = (int)time(nullptr);
    // random を 10 で割って、その余りを変数 answer に代入する
    // 10 で割った余りなので、answer には「0～9」のどれかが入っている
    int answer = t % 10;

    printf("%d\n", answer);

    // キーボードで押した数を変数 input に代入する
    int input;
    scanf_s("%d", &input);

    // ランクを保持する変数を宣言する
    int rank = 0;

    // answer に入っている数値と、input に入っている数値が一致していれば「一致！」と表示する
    if (answer == input)
    {
        rank = 2;
        printf("一致！\n");
    }
    // answer に入っている数値より、input に入っている数値の方が大きければ「大きい！」と表示する
    else if (answer < input)
    {
        printf("大きい！\n");
    }
    // answer に入っている数値より、input に入っている数値の方が小さければ「小さい！」と表示する
    else if (answer > input)
    {
        printf("小さい！\n");
    }

    // まずは answer と input の数値の差を計算する
    int sub = answer - input;
    if (sub > -3 && sub < 3 && sub != 0)
    {
        rank = 1;
        printf("惜しい！\n");
    }

    // rank に入っている数値に合わせて表示を変える
    switch (rank) 
    {
    case 2:
        printf("ランクS\n");
        break;
    case 1:
        printf("ランクA\n");
        break;
    default:
        printf("ランクB\n");
        break;
    }

    return 0;
}
```
