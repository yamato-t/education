# **変数**  

hello world プログラムを作成した。  
それを通じて順次処理を把握できたと思われる。

ただ、前のプログラムだと、予め指定した文字等を表示するだけしかできない。  
当然ながらそのようなプログラムだけでは、複雑なアプリケーションを作る事もできない。  

そこで今回は、人の入力によって結果が変わるプログラムを作成する。  
それを実現する為には、 <strong><span style ="color: #ff0000;">変数</span></strong> という物を知らなければならない。

今回は <strong><span style ="color: #ff0000;">変数</span></strong> とその利用方法の理解を目標に進めていく。

---

## **サンプルプログラム作成**

まずは一度コーディングする。  
前の授業で作成したプログラムを作ってほしい。  

```c++
#include <stdio.h>

int main() {
	printf("Hello world\n");
	return 0;
}
```

これを半角数字の「1」が表示される様に変更する。

```c++
#include <stdio.h>

int main() {
	printf("1\n");
	return 0;
}
```

では、ここからさらに、  
 <strong><span style ="color: #ff0000;">キーボードで押された任意の整数を表示したい</span></strong> となった場合はどうすれば良いか。  
例えば、5が押されれば5を表示し、8が押されれば8が表示されるなど。

その実現の為に必要な処理は何なのかを考える。  

- 任意の数字を表示する処理が必要
- キーボードで押された数字を取得する処理が必要

まずは <strong>任意の整数を表示する</strong> に着目する。

数字を表示しているのはここ
```c++
	printf("1\n");
```
これでは1という整数しか表示できない処理だとわかるはず。  
 **任意の整数を表示する** というのは、  
 状況によって「2」だったり「3」だったり「4」だったりに変わらないといけない。  
ただ、この書き方だとそれは実現できない。  
なぜならコードに「1」と書いてしまっているから。

## **変数宣言と利用**

こんな時に <strong><span style ="color: #ff0000;">変数</span></strong> を利用する。  
<strong><span style ="color: #ff0000;">変数</span></strong> とは、ある指定した値を入れておける箱の事を言う。

変数を利用した形に変える。

```c++
#include <stdio.h>

int main() {
	int number;
	number = 1;
	printf("%d\n", number);
	return 0;
}
```
ビルドして実行すると「1」と表示される。

変わったのは下記の部分。
```c++
	int number;
	number = 1;
	printf("%d\n", number);
```

このうち、一行目
```c++
	int number;
```
これが変数を用意している箇所になる。  
`number` が変数。  
このように変数を用意することを <strong><span style ="color: #ff0000;">変数を宣言する</span></strong> という。

二行目
```c++
	number = 1;
```
ここで変数に１という数を入れている。  
変数に数（値）を入れる事を <strong><span style ="color: #ff0000;">代入する</span></strong> という。

最後の行
```c++
	printf("%d\n", number);
```
変数に入っている数字を表示している。  
ではここから、10 と表示させたい場合はどう変更すれば良いか。  

### **変数に代入する際の注意**

```c++
	number = 10;
	number = 20;
```

変数は、一つにつき一つの値しか入れられない。  
**新しく代入すると、前に代入されていた値は分からなくなる**。

### **変数を宣言する際の注意**

**変数名は被ってはならない**。  
同じ名前の変数を宣言すると、変数の区別がつかなくなるのでコンパイルエラーになる。  

```c++
#include <stdio.h>

int main() {
	int a = 1;
	int a = 2; // コンパイルエラー

	return 0;
}
```


### **キーボード入力した値を変数に代入**

次に <strong>キーボードで押された整数をプログラムに教える</strong> 点に着目する。  
そもそもキーボードで押された数を知る方法だが、  
これは `printf` の様に機能が用意されている。
```c++
#include <stdio.h>

int main() {
	// 変数「number」を宣言する
	int number;	
	// 変数「number」にキーボードで押下した数値を代入する
	scanf_s("%d", &number);
	return 0;
}
```

では、ここまでの内容を整理して、  
 <strong><span style ="color: #ff0000;">キーボードで押された任意の整数を表示する</span></strong> というプログラムを作ること。

## **変数の型**
```c++
#include <stdio.h>

int main() {
	int number;
	number = 1.2;
	printf("%d\n", number);
	return 0;
}
```

今のプログラムで、小数を代入した際にどうなるか試してみてほしい。  
例えば、1.2 を入れてみたらどうなるか。

実は変数には <strong><span style ="color: #ff0000;">種類</span></strong> が存在する。  
この種類の事を <strong><span style ="color: #ff0000;">型</span></strong> という。  
変数は箱だと説明したが、値を正確に扱いたい場合、  
<strong><span style ="color: #ff0000;">箱とそこに入る値の型を一致させないといけない</span></strong> 。

今回、小数を入れても結果が1になっていた。  
これは宣言している変数の型が「整数」を扱う物になっているから。

### **変数型の指定**

**型は変数を宣言した時に決まる**。
```c++
	int number;
```
`int` が型にあたる。
int は integer の略。

これを小数を扱える型に変える。
```c++
	float number;
```
これで小数を扱う変数にできる。  
これでビルドして実行してみるとどうなるか。  

とんでもない数値が出ているはず。  
実は変更しなければならない箇所がもう一か所ある。
```c++
	printf("%d\n", number);
```
結論からいうと、`%d` で、この表記にはちゃんと意味がある。  
変数の中に入っている値を「どの型の値として表示するか」という事が、実はここで決められている。

`%d` というのは「変数の中に入っている値を『整数』として扱う」という意味になる。  
つまりここを「『小数』として扱う」という意味の表記に変えないといけない。

どうするべきか調べてみほしい。  
ここまでできたら<strong>キーボードで押された任意の小数を表示する</strong>プログラムを作ってみてほしい。

## **文字の扱い**
少し特殊な例として、 <strong><span style ="color: #ff0000;">文字</span></strong> に関して説明する。  
これまで変数に数値を代入していたが、型によっては文字を代入する事も可能。

```c++
	char c = 'a';
	printf("%c\n", c);
```
文字は `char型` に代入する。

そのまま文字を代入しようとすると、  
コーディング上の文字と判断が付かず、  
コンパイルエラーになる。 

そこで、文字を代入する場合は、「`（コーテーションマーク）」で囲う事で、  
「これは変数に代入する文字である」という事を明確にしている。  

文字を扱いたい場合は記号が必要という事を把握しておくこと。

## **四則演算**

指定した型の任意の値をプログラムで表示させるという事ができるようになったはず。  
ここからは実用的なアプリケーションとして計算機を作ってみる。

プログラムは、四則演算を行う事が可能。  
所謂、足し算、引き算、掛け算、割り算。  
これらはかなり直感的にコーディングできる。

```c++
#include <stdio.h>

int main() {
	// 1 + 1 の結果を変数「add」に宣言と同時に代入する
	int add = 1 + 1;
	// 1 - 1 の結果を変数「sub」に宣言と同時に代入する
	int sub = 1 - 1;
	// 1 x 1 の結果を変数「mul」に宣言と同時に代入する
	int mul = 1 * 1;
	// 1 / 1 の結果を変数「div」に宣言と同時に代入する
	int div = 1 / 1;

	// 2 + 3 x 4 の結果を変数「ans」に宣言と同時に代入する
	// この時に気を付けてもらいたいのが、計算の順番
	// 掛け算と割り算は足し算と引き算より先に計算される
	// 2 + 3 を先に計算したい場合は（）で括って (2 + 3) * 4 にする
	int ans = 2 + 3 * 4;
}
```

では、ここまで学んだ事を踏まえた上で、  
<strong>キーボードで押した二つの整数を足し算して結果を表示するアプリケーション</strong> を作成してほしい。

## **型変換**
下記のプログラムではどの様な結果になるか。

```c++
#include <stdio.h>

int main() {
	int a = 1;
	int b = 2;

	float answer = a / b;
	printf("%f\n", answer);

	return 0;
}
```
0.5となりそうだが、結果は 0.0 になる。  

```c++
	float answer = a / b;
```
`a / b` が両方とも整数型なので、結果がゼロになってしまっている。  
ゼロを `answer` に代入している事になる。

0.5 の結果を得る為には、` a や b ` を小数として扱わないといけない。  
こういった時、変数の型を変換させる事ができる。

```c++
	float answer = (float)a / (float)b;
```
変換したい型を()で括ると、その後ろに続く変数の型を変換する事ができる。  
これを <strong><span style ="color: #ff0000;">キャスト</span></strong> と呼ぶ。

変数に入っている値を別の型の変数に代入したりする場合に必要になるので、
覚えておくこと。

## **変数が使える範囲**

変数には「利用できる範囲」がある。

### **ローカル変数**

```c++
#include <stdio.h>

int main() {

	{
		int a = 1;
		{
			a = 2; // コンパイルエラーにはならない
		}
	}
	a = 2; // こちらはコンパイルエラー

	return 0;
}
```

変数は基本的に**宣言した波かっこの中でしか使えない**。  
変数には使える範囲があり、波かっこがその範囲を示している。  
波かっこの内側であれば、更に波かっこがあっても利用できる。

波かっこの内側で宣言された変数は**ローカル変数**と呼ばれる。

### **グローバル変数**

全ての波かっこの外側に宣言する事も可能。  

```c++
#include <stdio.h>

int a = 1; // 全ての波かっこの外側で宣言

int main() {

	{
		{
			a = 2; // コンパイルエラーにはならない
		}
	}
	a = 2; // コンパイルエラーにはならない

	return 0;
}
```

全ての波かっこの外側で宣言された変数は、ファイル全体で利用できる。  
この変数は**グローバル変数**と呼ばれる。  

但し注意として、グローバル変数の宣言は出来ても、  
利用は何らかの波かっこの中でしか出来ない。

```c++
#include <stdio.h>

int a = 1; // 全ての波かっこの外側で宣言
a = 3; // コンパイルエラーになる

int main() {

	{
		{
			a = 2; // コンパイルエラーにはならない
		}
	}
	a = 2; // コンパイルエラーにはならない

	return 0;
}
```

## **更新出来ない変数**

変数宣言時に `const` というキーワードを付ける事で、  
その変数の内容を上書き出来ない様にさせる事が可能。

```c++
// const 変数の「valueA」を宣言
const int valueA = 10; // 初期値の設定が必要

// 「valueA」を更新する
valueA = 1; // コンパイルエラーになる
```

現時点で何に使うのか想像しにくいかと思うが、  
<span style="color: red;">**更新させない**</span>という状況が後々メリットになる事もあるので、  
覚えておいてほしい。

## **ハードウェア寄りの話**

変数を利用して、プログラム内で値を保持、利用する方法を学んだ。  
それに関して、ハードウェア寄りで少し理解を深めていく。

変数は際限なく宣言できるわけではない。  
言葉で説明されてもピンと来ないと思われるので、  
コンピュータの中に大きな土地がある絵をイメージしてみること。  

仮に `int` 型の変数が宣言されたとする。

変数が宣言されると、その土地の一部が区切られ「レンタル中」となる。  
変数に値を代入すると、このレンタル中の土地にその値が保存される。  
これは宣言した変数用としてレンタルされているので、  
当然、他の変数が利用することはできない。

「土地の一部が区切られる」といったが、  
区切られる開始場所や範囲はどうやって決まっているか考えてみてほしい。

- **開始場所**  
勝手に決まる
- **範囲**  
宣言時に決まる  

先ほど変数の型の話をしたが、型に応じて範囲が決まっている。  
整数型の場合は、指定の開始場所から４つ分レンタルする、みたいな感じ。

ちなみに「範囲がある」という事から勘づいている方もいると思うが、  
保存できる値には限界が存在する。  

ここまで比喩で表していた事を正確にいうと、  

- 土地の事を**メモリ**  
- 場所の事を**アドレス**  
- 範囲の事を**サイズ**

と呼ぶ。

纏めると  
「変数を宣言すると、特定のアドレスから、変数の型に対応するサイズ分メモリを消費する」  
という言い方ができる。

なお、ここで挙げているメモリとは、主記憶装置つまりメインメモリと呼ばれる物の事を指す。  
<strong><span style ="color: #ff0000;">アプリケーションがデータを操作する為の作業領域</span></strong>と覚えておくこと。  

メモリにはヒープやスタック等、色々種類があるが、  
まずは「メモリ」という大雑把な理解で問題ない。

## **サイズに関して**

変数のサイズは「バイト長」になっている。  
バイトという単位を知る為には、  
まずビットという単位を知らなければならない。

コンピュータでは、色々な情報を扱える。  
写真や音楽、ゲームなど色々な情報をPCやスマートフォンに入れられていると思うが、  
実はそれらの情報は、全て <strong><span style ="color: #ff0000;">０と１の集合</span></strong> で出来ている。  

 <strong><span style ="color: #ff0000;">０と１しか入らない箱</span></strong> があり、  
 それが膨大に並ぶ事で、一つの情報が出来上がっているイメージ。

ここでいう、<strong><span style ="color: #ff0000;">０と１しか入らない箱</span></strong> が、  
コンピュータにおける最小単位であり、 <strong><span style ="color: #ff0000;">ビット</span></strong> と呼ばれている。

このビットが 8 個並ぶと、1 バイトになる。  
int 型は 4 バイトなので、ビットにすると、32 ビットになる。  
32個の 0 か 1 の並びで情報を持つことができる、という事。  

0 か 1 の並びは、そのまま二進数で表す事ができる。  
二進数はビットの値を扱う上で進数の理解は必須。
