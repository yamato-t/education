# **順次処理**

---

プログラムは<span style ="color: red;">**必ず上から下に向かって順に処理される**</span>。  
まずは下記プログラムコードを手元で記述してほしい。

```c++
#include <stdio.h>

int main() {
	printf("Hello world\n");
	return 0;
}
```

---

## **各記述の解説**


```c++
#include <stdio.h>
``` 

「おまじない」として書くと最初は教えられるが、  
これは <span style ="color: red;">**他の箇所で用意されている機能を利用できるようにする**</span> 為に書く。  
`stdio.h`　というファイルがあり、そこに用意されている機能を、  
その下にあるプログラム本体で利用できるようにする為に書いている。  

次がこのプログラムの本体になる。

```c++
int main() {
	printf("Hello world\n");
	return 0;
}
``` 

プログラムをコンパイルし実行した際 <span style ="color: red;">**main関数**</span> と呼ばれる箇所の頭から、  
実際に処理が行われる。  

 <span style ="color: red;">**main関数**</span> がどこかというと、  
```int main() { ``` の行から  
```}```の行までがそれにあたる。

「頭から処理が行われる」事から、  
実行した際に最初に処理される記述は下記の  

```c++
	printf("Hello world\n");
```

となる。  
これがコンソール画面に文字を表示している記述。

次の  

```c++
	return 0; 
```

は main 関数をここで終わらせるという意味の記述になる。  
 main 関数の終わりは、<span style ="color: red;">**アプリケーション全体の終了を意味する**</span>。

---

## **順次処理の挙動確認**

先ほどの例の下記の部分を

```c++
	printf("Hello world\n");
```

↓ に変更する

```c++
	printf("Hello world\n");
	printf("Hello world 2\n");
```

結果を確認すると

```
Hello world
Hello world 2
```

といった様に次の行に一文追加されていると思う。

では、記述の順を逆にするとどうなるか
```c++
	printf("Hello world 2\n");
	printf("Hello world\n");
```

表示される内容も逆になる。

```
Hello world 2
Hello world
```

ここからわかるように、プログラムは上から順に処理を行っている。

これが <span style ="color: red;">**順次処理**</span> であり、  
プログラムの基本構造の一つとなる。
