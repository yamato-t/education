# **配列**

---

今まで色々な情報を保存する為、変数を利用してきた。  
変数には整数や小数といった情報を保存し利用する事ができるが、  
一つの変数には一つの情報しか保持できないようになっている。

---

## **複数の情報を扱う**

ある四名の年齢のみを順に表示するプログラムを作成するとする。

- 一人目　：　15歳
- 二人目　：　32歳
- 三人目　：　11歳
- 四人目　：　48歳

試しに作ってみる。  
「引数で受け取った年齢を表示する関数」を作って利用するようにしてほしい。

```c++
#include <stdio.h>

// ---------------------------------------------------------
/**
 * @brief   年齢を表示する関数
 * @param   age 年齢
 */
void DispAge(int age) 
{
    printf("年齢は %d です\n", age);
}


// ---------------------------------------------------------
/**
 * @brief	main 関数
 */
int main() 
{
    DispAge(15);
    DispAge(32);
    DispAge(11);
    DispAge(48);

    return 0;
}
```

ここで考えてほしいのだが、  
年齢を表示したい人数が 100 人いた場合、どう表示させるか？

```DispAge``` を 100 回繰り返すという事から、  
**反復処理を使うといいんじゃないか**と思い浮かべた方もいるかもしれない。

その考えは正しいが、引数として渡される年齢はバラバラになっているので、  
単純に反復処理のみでは上手く対応出来ない。

こういった時に <span style ="color: red;">**配列**</span> を利用すると便利。

---

## **配列とは**

<span style ="color: red;">**配列**</span> とは、<span style ="color: red;">**同じ型の変数が複数個繋がっている物**</span> の事を指す。

配列は変数という箱が複数個繋がっているイメージで問題ない。  
配列をプログラムで宣言する。

```c++
// int 型の配列を宣言する
int ages[];
```

`[ ]` が付くようになった。  
ただし、これでは不十分でコンパイルエラーになる。

配列は「箱が複数個繋がっている物」と説明した。  
つまり「何個繋がっているのか」という事も、宣言の際に指定しなければならない。  
個数は `[ ]` の中に記述する。

```c++
// int 型の配列を宣言する
int ages[4];
```

これで 4 個分の `int` 型の変数を配列の形で宣言した。  
この様に宣言時に指定する箱の個数を <span style ="color: red;">**要素数**</span> という。

---

## **配列への代入と取得**

配列は箱が繋がっているイメージと話したが、  
各箱には先頭を 0 として順に番号がふられている。

この <span style ="color: red;">**0 番から始まっている**</span> という点は重要なので、  
しっかりと覚えておいてほしい。

今宣言した配列は 4 つの箱が繋がっている状態になっている。  
ではそれぞれに情報を入れてみる。

```c++
// 要素数が 4 の int 型の配列を宣言する
int ages[4];

// 配列の先頭（0 番）に情報を入れる
ages[0] = 15;

// 次（ 1 番）に情報を入れる
ages[1] = 32;

// 次（ 2 番）に情報を入れる
ages[2] = 11;

// 次（ 3 番）に情報を入れる
ages[3] = 48;

// 箱は 4 つしかないため、ages[4] = ?? はやってはいけない
```

注意すべきは `ages[4] = ??` ができない点。  
箱は 4 つしかない。  
先頭が 0 から始まる為 `ages[4]` は確保していない  
5 つ目の箱を指定する事になってしまう為。

箱から情報を取り出す時も同じように番号を指定する。

```c++
// 配列の先頭（0 番）の情報を取得する
int val = ages[0];

// 箱は 4 つしかないため、int val = ages[4] はやってはいけない
```

情報を入れる際や取り出す際に指定する `[ ]` の中の番号の事を  
<span style ="color: red;">**配列の添え字**</span> という。

### **変数による添え字**

添え字には変数を利用する事ができる。

```c++
// 番号を変数に代入する
int index = 1;

// index 変数を配列の添え字に利用する
// この場合、1 番の情報を取得する事が出来る
int val = ages[index];
```

---

## **配列の初期化**

配列を宣言した後、添え字を指定して値を入力していたが、  
通常の変数と同じように、宣言と同時に代入（初期化）する事もできる。

```c++
// 要素数が 4 の int 型の配列を宣言と同時に初期化する
int ages[4] = {
    15, // [0] に入る情報
    32, // [1] に入る情報
    11, // [2] に入る情報
    48, // [3] に入る情報
};
```

`=` でつなげた後に波カッコ（`{}`）で括り、  
その中にカンマ区切りで値を入れていく。  
カンマで区切る度に添え字が一つ増え、  
その添え字が指す箱に値が入る。

また、配列を宣言と同時に代入する場合のみ、  
カギカッコの中の個数を省略する事ができる。

```c++
// int 型の配列を宣言と同時に初期化する
// 初期化している個数で配列の要素数が決まる
int ages[] = {
    15, // [0] に入る情報
    32, // [1] に入る情報
    11, // [2] に入る情報
    48, // [3] に入る情報
};
```
