# **配列の次元**

横に繋がっている箱のイメージで配列を捉えてもらっていたが、  
こういった配列は横に繋がるだけの **一次元配列** と呼ばれる。

一次元があるという事は <span style ="color: red;">**多次元配列**</span> も存在する。  
横と縦に繋がる配列は **二次元配列** となる。

```c++
// int 型の二次元配列（ 横に 2 個並んだものが縦に 3 重なる ）を宣言する
int array[3][2];
```

配列のイメージはこの様になる。

| | |
| --- | --- |
| [0][0] | [0][1] |
| [1][0] | [1][1] |
| [2][0] | [2][1] |

横に 2 個繋がった箱が、縦に 3 つ重なっている様になっている。  
この多次元配列は 2 × 3、つまり要素数が 6 である事を意味している。

## **多次元配列の初期化**

この多次元配列の中身を全て 1 で初期化するとなった場合、  
どの様にすれば良いだろうか？

手元でやってみてほしい。

二通りできると思う。  
一つは `for` ループ、もう一つは宣言と同時に初期化。  
両方やってみてほしい。

```c++
// 宣言と同時に初期化
int array[3][2] = {
    {1, 1}, 
    {1, 1},
    {1, 1},
};

// for ループで初期化
for (int y = 0; y < 3; ++y) {
    for (int x = 0; x < 2; ++x) {
        array[y][x] = 1;
    }
}
```
