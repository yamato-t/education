# **テキストファイルの読み書き**

---

## **ファイルの開き方**

ファイルを開く際に利用する関数

```c++
// ファイルパスとモードを指定して、FILE構造体ポインタを取得する
// 戻り値は成否（成功時は0、失敗時はエラーコード）
fopen_s(FILE構造体ポインタのポインタ, ファイルパス, モード)
```

---

## **ファイルの閉じ方**

ファイルを閉じる際に利用する関数

```c++

// FILE構造体ポインタの操作を終了してファイルを閉じ、その成否を取得する
// 戻り値は成否（成功時は０、失敗時はEOF）
fclose(FILE構造体ポインタ)

```

---

## **ファイルの読み込み方**

ファイルを読み込む時に利用する関数（以下は一例）

```c++

// ファイルから 改行 or 指定した文字数-1 or ファイル終端（EOF）がくるまで文字列を読み取る
// 読み取りの開始位置は、FILE構造体側で制御されており、読み取りが行われると、読み取った分だけ開始位置を進める
// 戻り値は文字列の格納場所（終端ならNULL）
fgets(文字列の格納場所, 読み込む文字数, FILE構造体ポインタ)

// ファイルからフォーマットを指定して文字列を読み取る
// 読み取りの開始位置は、FILE構造体側で制御されており、読み取りが行われると、読み取った分だけ開始位置を進める
// 戻り値は入力個数（終端ならEOF）
fscanf_s(FILE構造体ポインタ, フォーマット指定文字列, 複数の読み込み先)

```

---

## **ファイルの書き込み方**

ファイルを書き込む時に利用する関数（以下は一例）

```c++

// 指定した文字列をファイルに書き込む
// 書き込みの開始位置は、FILE構造体側で制御されており、書き込みが行われると、書き込んだ分だけ開始位置を進める
// 戻り値は成否（成功ならゼロ以上、失敗ならEOF）
fputs(書き込む文字列, FILE構造体ポインタ)

// 指定したデータの内容を、フォーマットを指定してファイルに書き込む
// 書き込みの開始位置は、FILE構造体側で制御されており、書き込みが行われると、書き込んだ分だけ開始位置を進める
// 戻り値は書き込んだ文字数
fprintf(FILE構造体ポインタ, フォーマット指定文字列, 複数の書き込みデータ)

```

## **テキストファイルの読み書きの例題**

### **読み込みの実装例①**

```c++

#include <stdio.h>

// エントリー関数
int main(void)
{
    FILE* fp = NULL;

    // プロジェクトからの相対パスで asset/sample.txt を読込モードで開く
    // 開くことに失敗した場合は、エラーを表示する
    if (0 == fopen_s(&fp, "asset/sample.txt", "r"))
    {
        // 読込先の文字配列
        char buffer[256] = {};

        // fgets で文字を配列に詰め込む
        // 読み取り開始の現在位置はfpが持ち、fgetsが成功すると、読み取った分だけ、fpが持つ読み取り開始の位置を進めてくれる
        // その為、戻り値が NULL (EOFに到達 or 読み取りに失敗)になるまで while で読み取りを繰り返すといった事が出来る     
        while (NULL != fgets(buffer, 256, fp))
        {
            // 詰め込んだ内容を表示する
            printf("%s", buffer);
        }

        // 閉じる
        fclose(fp);
    }
    else 
    {
        printf("ファイルがありません\n");
    }

    return 0;
}
```


### **読み込みの実装例②**


```c++

#include <stdio.h>

// エントリー関数
int main(void)
{
	FILE* fp = NULL;

	// プロジェクトからの相対パスで asset/sample.txt を読込モードで開く
	// asset/sample.txt には 「1 aaaa 1.7」といった内容が入っている
	// 失敗した場合はエラー表示
	if (0 == fopen_s(&fp, "asset/sample.txt", "r"))
	{
		// 読込先の整数変数
		int no = 0;
		// 読込先の文字配列
		char string[256] = {};
		// 読込先の浮動小数変数
		float value = 0;

		// fscanf_s でフォーマットを指定して読み取る
		while (EOF != fscanf_s(fp, "%d %s %f", &no, string, 256, &value))
		{
			// 読み込んだ内容を表示する
			printf("%d,%s,%1.2f\n", no, string, value);
		}

		// 閉じる
		fclose(fp);
	}
	else 
	{
		printf("ファイルがありません");
	}

	return 0;
}
```

### **書き込みの実装例①**

```c++

#include <stdio.h>

// エントリー関数
int main(void)
{
	FILE* fp = NULL;

	// プロジェクトからの相対パスで asset/sample.txt を書込モードで開く
	// asset フォルダが無いと失敗する事に注意する
	if(0 == fopen_s(&fp, "asset/sample.txt", "w"))
	{
		// 文字列を書き込む
		fputs("text write", fp);

		// 閉じる
		fclose(fp);
	}
	else {
		printf("ファイルを開けませんでした");
	}

	return 0;
}
```


### **書き込みの実装例②**

```c++

#include <stdio.h>

// エントリー関数
int main(void)
{
	FILE* fp = NULL;

	// プロジェクトからの相対パスで asset/sample.txt を書込モードで開く
	if (0 == fopen_s(&fp, "asset/sample_write.txt", "w"))
	{
		// 文字を入力する
		// ※スキャン集合を利用して改行だけ読み取らない、つまり、スキャンを終えるのは改行が来た時だけ
		char buffer[100] = {};
		scanf_s("%[^\n]", buffer, 100);

		// 数字を入力する
		int no = 0;
		scanf_s("%d", &no);

		// 数字を入力する
		float value = 0;
		scanf_s("%f", &value);

		// フォーマットを指定して書き込む
		fprintf(fp, "%s, %d, %f", buffer, no, value);

		// 閉じる
		fclose(fp);
	}
	else
	{
		printf("ファイルを開けませんでした");
	}

	return 0;
}
```
