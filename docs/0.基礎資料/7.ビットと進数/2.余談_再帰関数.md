# **再帰関数**

---

## **べき乗を計算するプログラム** ##

べき乗を計算するプログラムを作成する際、ループ処理だけではなく、  
自分自身を呼び出す「**再帰**」という方法でも実現が可能。  
この様な関数は<span style="color: red;">**再帰関数**</span>と呼ばれる。

```c
#include <stdio.h>

// べき乗を計算する関数(ループ)
int loopExp(int base, int n)
{
	int res = 1;

	for (int i = 0; i < n; ++i)
	{
		res *= base;
	}

	return res;
}


// べき乗を計算する関数（再帰）
int recursionExp(int base, int n)
{
	int res = 1;

	if (n != 0) 
	{
		// n がゼロにならない限り、自身を呼び続ける
		res = base * recursionExp(base, n - 1);
	}

	return res;
}


// エントリー関数
int main(void)
{
	int loop = loopExp(2, 4);
	printf("%d\n", loop);

	int recursion = recursionExp(2, 4);
	printf("%d", recursion);

	/*
		recursionExp(2, 4)は階層を潜っていく様に呼び出される

		1.  recursionExp(2, 4)                 
		2.  2 * recursionExp(2, 3)             
		3.  2 * 2 * recursionExp(2, 2)         
		4.  2 * 2 * 2 * recursionExp(2, 1)     
		5.  2 * 2 * 2 * 2 * recursionExp(2, 0) 
		6.  2 * 2 * 2 * 2 * 1　

		以降は潜った階層を一段ずつ上っていく様に return される

		7.  2 * 2 * 2 * 2　
		8.  2 * 2 * 4　
		9.  2 * 8　
		10. 16　

	*/

	return 0;
}
```
