# **ビット単位の操作**

---

## **ビット演算子**  
各ビットに対して演算を行う際に指定する演算子。  

|  名称  |  演算子  |　内容 | 例 |
| ---- | ---- | ---- | ---- |
|  AND  | & | 双方が「1」なら「1」になる | (1001 & 0011) = 0001  |
|  OR  | │ | どちらかが「1」なら「1」になる | (1001 │ 0011) = 1011  |
|  NOT  |  ~  | 「0」は「1」に、「1」は「0」になる | (~1001) = 0110  |
|  XOR  |  ^  | 双方が異なる値なら「1」になる | (1001 ^ 0011) = 1010  |
|  左シフト  |  <<  | ビットを指定した数だけ左にずらす | (1001 << 1) = 0010  |
|  右シフト(※)  |  >>  | ビットを指定した数だけ右にずらす | (0011 >> 1) = 0001  |

### **符号付き右シフトの注意**
シフト後、空いたビットは基本的に0で埋められるが、  
**符号付きの2進数を右シフトする場合は、**  
**最上位ビットに入っていた値(同じ符号)で埋められる。**

例) 符号付き 1001 を右に２シフトする
　　→　(1001 >> 2) = 1110

---

## **& と &&**  
プログラムで「&&」と記述する事があるが、これは「**論理演算子**」であり、  
「真か偽か？」を表す演算子で、ビット演算子である「&」とは全く違う物になる。  
「|」と「||」も同じ。

## **ビット操作のプログラム利用例**

### **状態チェック処理**
ビット操作をゲームプログラムで利用する例として、  
キャラクターの状態チェックを挙げる

```c
#include <stdio.h>

// キャラクターのステータス異常種類
// これはビットシフト数として利用する
enum Status
{
	Poison,			// 毒
	Burn,			// 火傷
	Freezing,		// 氷結
	Paralysis,		// 麻痺
	Confusion,		// 混乱
};

// エントリー関数
int main(void)
{
	// ステータスを保持する変数
	// ステータスは８ビット分まで保持する事が出来る
	unsigned char statusFlag = 0;

	// ステータスを氷結にする
	// ビット演算子で氷結を指すビットを「１」にする
	statusFlag = (1 << Freezing);

	// この時の statusFlag を2進数にすると以下のようになる
	// ---------------------------------
	// | 0 | 0 | 0 | 0 | 0 | 1 | 0 | 0 |
	// ---------------------------------

	// さらにステータスに混乱を加える
	// 「=」ではなく「|=」にしている点に注意
	statusFlag |= (1 << Confusion);

	// この時の statusFlag を2進数にすると以下のようになる
	// ---------------------------------
	// | 0 | 0 | 0 | 1 | 0 | 1 | 0 | 0 |
	// ---------------------------------

	// ステータスが氷結かどうかをチェックする
	if (statusFlag & (1 << Freezing))
	{
		printf("氷結です");
	}

	return 0;
}
```
