# **反復処理**

---

## **反復処理とは**

分岐処理を利用して数当てゲームを作成した。  
リザルトランクの表示など、ある程度ゲームらしさが出てきたが、  
現状では一度入力したら、どんな結果であろうと終了してしまう。  

これではゲームとは呼べない。  
ゲームオーバーになれば、コンテニューを選べるようにする必要があるし、  
何度も挑戦できるよう、処理を繰り返せる形にしなければならない。  

今回はその<span style ="color: red;">**繰り返し、反復処理**</span>について学ぶ。  
順次処理、分岐処理と同じく、反復処理は**プログラム基本構造の一つ**。  

---

## **サンプルプログラムの作成**

反復処理の説明に入る前に、数当てゲームとは別で、  
新規のコンソールプロジェクトを作成し、以下のサンプルを用意する。

```c++
#include <stdio.h>

int main() {

    int count = 0;
    printf("現在のcount は %d です\n", count);

    return 0;
}
```

このプログラムを実行すると、当然以下のように表示される。

```
現在のcount は 0 です
```

では、以下のように表示させるにはどうすればよいか？

```
現在のcount は 0 です
現在のcount は 1 です
現在のcount は 2 です
現在のcount は 3 です
現在のcount は 4 です
現在のcount は 5 です
現在のcount は 6 です
現在のcount は 7 です
現在のcount は 8 です
現在のcount は 9 です
```

手元のプログラムを書き換えてみてほしい。

できたら次の問題。  
今度は 0 から 99 まで、以下のように表示させるにはどう書けばよいか？

```
現在のcount は 0 です
現在のcount は 1 です
・
・
・
現在のcount は 99 です
```

順番に記述していけばもちろん実現できる。  
しかし 100 行ならまだしも、10000 まで表示しろと言われたら無理がある。  

ここで着目してほしいのは、数がいくら増えようとも、  
以下のコードは一切変化していないという点。

```c++
    printf("現在のcount は %d です\n", count);
```

変わっているのは、`count` 変数に代入されている数値のみ。  

やりたいことは「0 〜 99 まで表示する」こと。  
そして、それを実現しているコードは全く同じ一行。  
この一行を書いておけば、あとは自動で 100 回繰り返してくれると助かる。  
そのために使えるのが <span style ="color: red;">**反復処理**</span> 。

反復処理を使うことで、一定範囲内のコードを条件に応じて繰り返すことが可能になる。

---

## **while 文**

### **基本構文**

まずはコーディング。

```c++
// count が 100 より小さい間は表示を繰り返す
while(count < 100)
{
    printf("現在のcount は %d です\n", count);
    count = count + 1;
}
```

`while` は<span style ="color: red;">**指定した条件で波かっこの処理を繰り返す**</span>キーワード。  
`()` の中に記述した条件が `true` である限り、波かっこ内の処理を繰り返す。  
条件の書き方は `if 文` と全く同じ。

各行の意味を整理すると、以下のようになる。

- <span style ="color: red;">**`while(count < 100)`**</span>
    - `count` が 100 より小さければ `true`、100 以上なら `false`
    - `true` の間、波かっこ内を繰り返す
- <span style ="color: red;">**`printf(...)`**</span>
    - 繰り返したい処理本体
- <span style ="color: red;">**`count = count + 1;`**</span>
    - `count` に 1 を加算し、`count` へ再代入する
    - `++count` や `count += 1` と書くこともできる

日本語に翻訳すると、  
「変数 `count` が 100 より小さい場合、波かっこの処理を繰り返す」  
という意味になる。  
この書き方を <span style ="color: red;">**while 文**</span> という。

### **無限ループに注意**

`count = count + 1;` の行を削除すると何が起きるか想像して欲しい。  
繰り返す条件が「count が 100 より小さい」である以上、  
`count` が増えない限り条件は永遠に `true` のまま。  
これが <span style ="color: red;">**無限ループ**</span>を引き起こす。

### **do while 文**

`while` キーワードを使った別の書き方として、<span style ="color: red;">**do while 文**</span> がある。

```c++
// count が 100 より小さい間は表示を繰り返す
do
{
    printf("現在のcount は %d です\n", count);
    count = count + 1;
}
while(count < 100);
```

`while 文` と `do while 文` の違いは**条件チェックのタイミング**。

| 種類 | 条件チェックのタイミング | 備考 |
|------|--------------------------|------|
| while 文 | 波かっこの処理より先 | 条件が最初から false なら一度も実行されない |
| do while 文 | 波かっこの処理の後 | 必ず最低 1 回は実行される |

日本語に翻訳すると、  
「波かっこの処理を行った後、変数 `count` が 100 より小さかった場合は繰り返す」  
という意味になる。

### **反復処理の考え方**

一見すると「処理が戻っている」ように見えるが、  
反復処理は波かっこの中を常に上から順に処理し、  
それをルールに則って一定数繰り返しているのみ。

「戻る」のではなく「もう一度、上から順に処理している」という認識を持つこと。  
**処理位置が戻ったとしても、それまでの処理が無かったことになるわけではない**。

## **for 文**

### **基本構文**

反復処理を行えるキーワードは `while` だけではない。

```c++
// count が 100 より小さい間は表示を繰り返す
for(count = 0; count < 100; count = count + 1)
{
    printf("現在のcount は %d です\n", count);
}
```

`for` も<span style ="color: red;">**指定した条件で波かっこの処理を繰り返す**</span>キーワード。  
`while` とは記述が異なるが、同じく反復処理を実現できる。  

`for` に続く `()` の中はセミコロン 2 つで 3 つに区切られており、  
それぞれ以下の役割を持つ。

| 位置 | 記述例 | 役割 |
|------|--------|------|
| 1 つ目 | `count = 0` | 初期化。繰り返し開始時に一度だけ実行される |
| 2 つ目 | `count < 100` | 条件。`true` の間繰り返す（`while` と同じ） |
| 3 つ目 | `count = count + 1` | 更新。波かっこの処理が終わるたびに実行される |

日本語に翻訳すると、  
「変数 `count` が 0 から始まり、`count` が 100 より小さい間、  
波かっこの処理を繰り返しながら、`count` に 1 ずつ足し続ける」  
この書き方を <span style ="color: red;">**for 文**</span> という。

### **さまざまな書き方**

`for 文` は省略による様々な書き方ができる。以下はその一例。

```c++
// 更新部分を省略し、波かっこ内でインクリメントする
for(count = 0; count < 100; /*なし*/)
{
    printf("現在のcount は %d です\n", count);
    ++count;
}
```

```c++
// 初期化も省略する（count がすでに 0 の場合）
for(/*なし*/; count < 100; /*なし*/)
{
    printf("現在のcount は %d です\n", count);
    ++count;
}
```

初期化を省略する場合、`for 文` 開始時の `count` の値によって  
繰り返し回数が変わる点に注意が必要。  
例えば `count` が 90 の状態で開始されれば、90 〜 99 の表示になる。

こうした問題を避けるため、繰り返し専用の変数を `for 文` 内で用意する方法もある。

```c++
// 繰り返しの条件のために変数 i を別で用意する
for(int i = 0; i < 100; ++i)
{
    count = i;
    printf("現在のcount は %d です\n", count);
}
```

`for 文` 内で宣言した変数は、波かっこ内でそのまま利用できる。

## **中断処理**

### **break による強制終了**

繰り返し処理の途中で、強制的に終了させたい場合も多々ある。  
例えば以下は意図的な無限ループの例。

```c++
// 延々と何らかの処理を繰り返す
while(true)
{
    // 何らかの処理
}
```

この繰り返しを、内部の処理結果を条件にして終了させたい場合、  
先週も登場した <span style ="color: red;">**`break`**</span> を使う。  
`break` は switch 文だけでなく、反復処理の終了にも使えるキーワード。

```c++
// 延々と何らかの処理を繰り返す
while(true)
{
    // 何らかの処理

    if(/*終了条件*/)
    {
        // 繰り返しの処理を終わらせる（無限ループを抜ける）
        break;
    }
}
```

`while 文` の例で示したが、`for 文` でも同様に使用できる。

繰り返し処理を強制終了させる場面は無限ループに限らず、  
今後ゲームを作る中で数えきれないほど登場する。  
今のうちにしっかりと理解を深めておくこと。

## **纏め**

最終的なコードはこのようになる。

```c++
/*
 * コンピュータが決めたランダムな数値を当てる簡易ゲームを作る
 */
#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main() {

    // コンピュータの時間（を表す値）を変数に代入する
    int t = (int)time(nullptr);
    // random を 10 で割って、その余りを変数 answer に代入する
    // 10 で割った余りなので、answer には「0～9」のどれかが入っている
    int answer = t % 10;

	// ゲームをクリアしたかどうかを保持する変数 clear を宣言して、false を代入する
    bool gameClear = false;

	// ゲームをクリアするまで繰り返す
    while (!gameClear) {

        printf("ゲーム開始！数値を入力してください！-----\n");

        // キーボードで押した数を変数 input に代入する
        int input;
        scanf_s("%d", &input);

        // ランクを保持する変数を宣言する
        int rank = 0;

        // answer に入っている数値と、input に入っている数値が一致していれば「一致！」と表示する
        if (answer == input)
        {
            rank = 2;
            printf("一致！\n");
        }
        // answer に入っている数値より、input に入っている数値の方が大きければ「大きい！」と表示する
        else if (answer < input)
        {
            printf("大きい！\n");
        }
        // answer に入っている数値より、input に入っている数値の方が小さければ「小さい！」と表示する
        else if (answer > input)
        {
            printf("小さい！\n");
        }

        // まずは answer と input の数値の差を計算する
        int sub = answer - input;
        if (sub > -3 && sub < 3 && sub != 0)
        {
            rank = 1;
            printf("惜しい！\n");
        }

        // rank に入っている数値に合わせて表示を変える
        switch (rank)
        {
        case 2:
			// rank に 2 が入っているときは「ランクS」と表示して、ゲームをクリアする
            printf("ランクS\n");
			gameClear = true;
            break;
        case 1:
            printf("ランクA\n");
            break;
        default:
            printf("ランクB\n");
            break;
        }

		// ゲームをクリアしていないときは「もう一度！」と表示する
        if (!gameClear)
        {
            printf("もう一度！\n\n");
        }
        else {
			printf("ゲームクリア！\n");
        }
    }

    return 0;
}

```
