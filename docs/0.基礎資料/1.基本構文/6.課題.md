## **課題①　複数教科の平均点と合否判定プログラムを作成してください**

5教科の点数を入力し、  
以下の処理を行うプログラムを作成してください。

- 5教科の点数（0〜100の整数）を順番に入力する
- 合計・平均点を小数点第1位まで表示する
- 以下の条件で合否を判定する
  - 平均点が60点以上、かつ1教科も40点未満がない → **合格**
  - 上記以外 → **不合格**（理由も表示する）

不合格の理由は、両方の条件に当てはまる場合はどちらも表示してください。  
教科数は固定（5教科）とし、配列は使用しないでください。

## **課題②　繰り返し計算できる電卓プログラムを作成してください**

2つの整数と演算子を入力して計算結果を表示し、  
以下の処理を行うプログラムを作成してください。

- 整数を2つと演算子（`+` `-` `*` `/`）を入力して結果を表示する
- 計算後に続けて計算するか確認し、`y` で継続・それ以外で終了する
- `/` の場合は0除算をチェックし、エラーメッセージを表示する
- 未知の演算子が入力された場合もエラーメッセージを表示する

## **課題③　指定した回数だけ Fizz Buzz を表示するプログラムを作成してください**

開始値・終了値を入力し、  
以下のルールで FizzBuzz を表示するプログラムを作成してください。

- 開始値と終了値（1〜1000の整数）を入力する
- 開始値から終了値まで順に以下を表示する
  - 3の倍数かつ5の倍数 → `FizzBuzz`
  - 3の倍数 → `Fizz`
  - 5の倍数 → `Buzz`
  - それ以外 → その数をそのまま表示する
- 開始値が終了値より大きい場合はエラーメッセージを表示する
- 範囲外の入力（1未満・1000超）もエラーメッセージを表示する

## **課題④　コインを使った両替プログラムを作成してください**

金額を入力し、硬貨の枚数が最小になるよう両替して表示する処理と、  
以下の処理を行うプログラムを作成してください。

- 1以上の整数（円）を入力し、各硬貨の枚数を表示する
  - 使用する硬貨：500円・100円・50円・10円・5円・1円
  - 枚数の計算は `/` と `%` を使って順番に求める
  - 0枚の硬貨は表示しない
- 続けて両替するか確認し、`y` で継続・それ以外で終了する
- 終了時に合計両替回数と合計金額を表示する

配列は使用しないでください。

## **課題⑦　複数ファイルで構成する両替プログラムを作成してください**

課題④の両替プログラムを、以下のファイル構成で再実装してください。

- `coin.h` / `coin.cpp`：両替ロジックに関する関数を定義する
  - 金額と硬貨の額面を受け取り、その硬貨の枚数を返す関数
  - 金額と各硬貨の枚数を受け取り、両替結果を表示する関数（0枚は表示しない）
- `summary.h` / `summary.cpp`：集計に関する関数を定義する
  - 合計両替回数と合計金額を受け取り、まとめを表示する関数
- `main.cpp`：ループ処理と入力処理のみを記述する

各ファイルは `#include` で適切に連携させてください。
