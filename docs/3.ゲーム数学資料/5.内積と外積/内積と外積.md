# **内積と外積**

## **内積**

以前の授業で「プレイヤー（矢印）が向いている方向を判定する」というプログラムを見て頂いた。  
ただ、実際のゲームでは「プレイヤーの視野角内にいれば」等の条件が追加される場合が大半。  

プレイヤーの向きから左右に「○○度の範囲にいるか」をどう計算するのか、だが、  
こういう状況では <span style ="color: #ff0000;"> **内積** </span>という物が利用される。

内積は<span style ="color: #ff0000;"> スカラー積 </span>あるいは<span style ="color: #ff0000;"> ドット積 </span> と呼ばれたりする。

内積の有用な特性として

- <span style ="color: #ff0000;"> **二つの単位ベクトルがなす角度が分かる** </span> 
- <span style ="color: #ff0000;"> **直線と点の最短距離を計算できる** </span> 

の二つが挙げられる。

![dot.png](https://bitbucket.org/repo/q7MKna4/images/2968231341-dot.png)


### **内積の計算式**


$\vec{A} = (A_{1}, A_{2}, A_{3})$ と $\vec{B} = (B_{1}, B_{2}, B_{3})$ で、  
二つのベクトルのなす角が θ の時の内積計算式  

$$\vec{A}\cdot\vec{B}=|\vec{A}||\vec{B}|\cos\theta=A_{1}B_{1}+A_{2}B_{2}+A_{3}B_{3}$$

式を見てもらうとわかる通り、内積の結果は「ある一つの値」になる。  
要するに向きや方向ではなく量を計算する物。  

---
## **Sampe 1**

単位ベクトル同士で内積を計算すると<span style ="color: #ff0000;"> **-1 ~ 1 の間の値になり cos として利用できる** </span> 。  
**二つの単位ベクトルがなす角度が分かる** という特性はここからきている。

内積の結果が

- <strong><span style ="color: #36b336;"> 1 の場合 </span></strong> ：全く同じ方向を向いているベクトル
- <strong><span style ="color: #36b336;"> 0 の場合 </span></strong> ：直交（90度）しているベクトル
- <strong><span style ="color: #36b336;"> -1 の場合 </span></strong>：正反対(180度)の方向を向いているベクトル

という事が言える。  

---
## **Sampe 2**

A は単位ベクトルにせず、B だけ単位ベクトルにする。  
その状態で内積を計算すると、  
B の方向に沿った A の大きさ 『 $|\vec{A}|\cos\theta$ 』が分かるようになる。  

別ベクトルに影を落とすというイメージが分かりやすい。  
この様に別ベクトルに沿った影を落とす事を<span style ="color: #ff0000;">**射影**</span>と呼ぶ。

![dot2.png](https://bitbucket.org/repo/q7MKna4/images/3858451429-dot2.png)

こんなの何に使う？と思われるかもしれないが、  
<span style ="color: #ff0000;"> **点と直線の最短距離** </span>を計算する時などに利用される。


---
## **外積**

内積に対して<span style ="color: #ff0000;"> **外積** </span>という物もある。  
内積と同じく二つのベクトルの関係性を表す結果を得られるが、  
外積の場合<span style ="color: #ff0000;"> **二つのベクトルと直交するベクトル** </span>を得る事が出来る。  
<span style ="color: #ff0000;"> **クロス積** </span>等と呼ばれたりもする。

外積は<span style ="color: #ff0000;"> **二つのベクトル両方に直交するベクトルが分かる** </span> という特性がある。

![cross.png](https://bitbucket.org/repo/q7MKna4/images/138296455-cross.png)


### **外積の計算式**

$\vec{A} = (A_{1}, A_{2}, A_{3})$ と $\vec{B} = (B_{1}, B_{2}, B_{3})$ の外積計算式  
$$\vec{A}\times\vec{B}=\begin{pmatrix}A_{2}B_{3}-A_{3}B_{2}\cr A_{3}B_{1}-A_{1}B_{3}\cr A_{1}B_{2}-A_{2}B_{1}\cr\end{pmatrix}$$

外積の大きさ $|\vec{A}\times\vec{B}|$   
$$|\vec{A}\times\vec{B}| = |\vec{A}||\vec{B}|\sin\theta$$ 

同一のベクトルで外積を計算するとゼロベクトルになる。  
なお、外積結果のベクトルの大きさは、  
二つのベクトルが成す四角形の面積と一致する。

---
## **Sampe 3**

例えば、あるターゲットの方向を向かせたい場合、  
角度は内積で計算できるが、**どの方向に回転すればいいのかはわからない**。  
仮に角度が 30 だったとして、  
プラスの方向に傾けるのかマイナスの方向に傾けるのかは状況によって変わる。

そこで外積を使う。  
外積で計算したベクトルは「回転軸」として利用できる。  
軸の向きによって、回転方向を任意に選ぶ事が出来る。


## **外積で平面の向きを知る**

二つのベクトルは四角形を形作るが、  
外積はその四角形の面の向きベクトルを表している。  
3D の描画を扱う際、よく<strong><span style ="color: #ff0000;"> ポリゴンの法線 </span></strong>という言葉と出くわすと思う。  
この法線が面の向きベクトルの事を指す為、覚えておいてほしい。
