# **運動**

---
## **等速運動**

座標に移動ベクトルを加える事で、新しい座標に移動させていた。  
単純にボタンを押している間動き続けていたが、  
<span style ="color: #ff0000;">**毎フレーム同じ移動量**</span>で移動していたはず。  

速さが一定の移動は<span style ="color: #ff0000;">**等速運動**</span>と呼ばれる。

今回はそういった等速運動ではなく、  
加速していく移動に関して学んで頂く。

---
## **ある方向に等速で移動する挙動**

「スペースを押すとx軸方向に移動量 0.1 で動き続け、壁にぶつかると元の位置に戻る」  
という処理を作成する。

### **移動時間の計算**

壁まで 30 の距離があるが、単位がないとわかりにくいので 30 m と考える。  
ここで「壁に到達するまでのカウントダウンを表示したい」となったらどうするか。  

壁にぶつかるまでの時間を計算して、発射と同時にカウントダウンする。

移動量は 0.1 m なので、  
何回移動すると 30m に到達するかを計算する。  

$30\hspace{2pt}\mathrm{m}\div 0.1\hspace{2pt}\mathrm{m/f}=300\hspace{2pt}\mathrm{f}$

300 回、つまり 300 フレームで到達する。  
60fps で考えると 5 秒かかる事になる。

スペースキーを押した瞬間に 5 秒からカウントダウンが始まって、  
残りゼロ秒になったら、壁にぶつかる事が分かる。

---
## **ある方向に加速しながら移動する挙動**

では加速する場合はどうなるか。  
移動量が毎フレーム 0.01 ずつ増える様に変更する。

一定の値で速度が増えていく or 減っていく運動の事を 
**<span style ="color: #ff0000;"> 等加速度運動 </span>** と呼ぶ。

### **移動時間の計算**

これでも壁にぶつかるまでのカウントダウンを表示する。

等速運動の場合、割るだけで時間が出るが、  
速度が変わり続けている今回は、同じ計算式は使えない。  

等加速度運動には下記の公式がある。

時間　　: $t\hspace{2pt}(\mathrm{f})$   
初速　　: $v_{0}\hspace{2pt}(\mathrm{m/f})$     
加速度　: $a\hspace{2pt}(\mathrm{m/f^2})$   
とする場合

- t フレーム経過後の速度 v を計算する式  
$v = v_{0} + at$
- t と t フレーム経過後の速度の差から a を計算する式  
$a=\frac{v-v_{0}}{t}$
- t フレーム経過後の変異（位置の変化量） x を計算する式  
$x=v_{0}t+\frac{1}{2}at^2$
- v と x の関係を指す式  
$v^2-{v_{0}}^2=2ax$  

これを利用し、  
壁にぶつかるまでの時間を事前に計算する。  

---
## **単位に関する注意点**

1秒は 60 フレームの為、単位を変えると数値も変わる。

初速　　$v_{0} = 0.1\hspace{2pt}\mathrm{m/f} = 6\hspace{2pt}\mathrm{m/s}$  
加速度　$a = 0.01\hspace{2pt}\mathrm{m/f^2} =  36\hspace{2pt}\mathrm{m/s^2}$  

距離は時間の単位を利用していないので同じ。  
これで計算しなおしてみてほしい。

余談として、仮に両方の単位が変わる場合は下記のようになる。  
$\mathrm{m/s^2}$ の単位を $\mathrm{km/h^2}$ に変える場合、  

距離の単位の違い：$1\hspace{2pt}\mathrm{m} = 0.001\hspace{2pt}\mathrm{km}$  
時間の単位の違い：$1\hspace{2pt}\mathrm{h} = 3600\hspace{2pt}\mathrm{s}$  
なので、  
$10\hspace{2pt}\mathrm{m/s^2} = 10 \times 0.001 \times 3600^2 = 129600\hspace{2pt}\mathrm{km/h^2}$

## **微分と積分**

一定の加速が加わり続けるだけで、  
到着するまでの時間や距離の計算が複雑になる。  

速度と時間を元に距離を計算する場合、等速だと単純に掛け算するだけだった。  
これをグラフで表すとこうなる。  
![運動1.png](https://bitbucket.org/repo/LbeA8Ky/images/3816919591-%E9%81%8B%E5%8B%951.png)  
この面積が距離を表している事になる。

では等加速度運動ではどの様なグラフになるか。  
よく見ると台形になっている事が分かる。  
![運動2.png](https://bitbucket.org/repo/LbeA8Ky/images/339686181-%E9%81%8B%E5%8B%952.png)  
(上底 + 下底) x 高さ / 2  が式になるが、  
これに情報をあてはめると、先程紹介した式の一つが出来上がる。

$x = (v_{0} + v) \times {t} \times \frac{1}{2}$  
$ \Leftrightarrow x = (v_{0} + (v_{0} + at)) \times \frac{1}{2}t $  
$ \Leftrightarrow x = (2v_{0} + at) \times \frac{1}{2}t $  
$ \Leftrightarrow x = v_{0}t + \frac{1}{2}at^2$  

では加速度自体が時間によって上下する場合はどうなるか。    

この状態で面積を出す事なんて出来るのか？となるのだが、  
一瞬毎の速度で進んだ距離を全て足し合わせる事で正確な距離を計算する事が出来る。  
これを**積分**と呼ぶ。  

対して、ある瞬間の位置の変化からその一瞬の速度を計算する事も可能。  
これを**微分**と呼ぶ   

![運動4.png](https://bitbucket.org/repo/LbeA8Ky/images/1651884459-%E9%81%8B%E5%8B%954.png)


先程利用したこの二つの式は微分と積分の関係にある。

$v = v_{0} + at$  
積分↓  ↑微分  
$x=v_{0}t+\frac{1}{2}at^2$
