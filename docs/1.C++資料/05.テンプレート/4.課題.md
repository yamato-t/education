# **課題**

## **課題①　各関数テンプレートを作成してください**

template.h を新規作成し、  
下記の関数テンプレートを定義も含めて記述してください。

- 同じ型同士を加算して返す関数
- 同じ型同士を減算して返す関数
- 同じ型同士を乗算して返す関数
- 同じ型同士を除算して返す関数

各テンプレート関数は main 関数で利用できるようにしてください。

## **課題②　各関数テンプレートを作成してください**

template.h に、  
下記の関数テンプレートを定義も含めて記述してください。

- 違う型同士を加算して、どちらかの型で返す関数
- 違う型同士を減算して、どちらかの型で返す関数
- 違う型同士を乗算して、どちらかの型で返す関数
- 違う型同士を除算して、どちらかの型で返す関数

各テンプレート関数は main 関数で利用できるようにしてください。
