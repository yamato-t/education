# **テンプレート特殊化**

---
## **特定の型だけ別の挙動にしたい**

テンプレートを用意することで、一つの記述で具体的な型に対応した処理が生成されるが、  
場合によっては<span style="color: red;">**特定の型だけ例外的に処理したい**</span>という状況が発生する。

```c++
#include <iostream>

// T 型の値を合成する関数テンプレート宣言、定義
template<class T>
void combine(T l, T r)
{
   T res = l + r;
   std::cout << res  << std::endl;
}

// エントリー関数
int main()
{
    combine(1, 2);
    combine(2.6f, 3.1f);

    return 0;
}


```

上記は値同士を合成して結果を出力する関数になる。  
ここに下記のような呼び出しを追加する。
```c++
    combine("1月23日", "晴れ");
```

日付と天気の文字列を合成して出力したいという事なのだが、  
```combine``` 関数の中で **加算** している為、意図した挙動にならず、  
それどころかコンパイルエラーになる。

特定の型の場合だけ、特殊な挙動を行わせたい。  
このような場合<span style="color: red;">**テンプレートの完全特殊化（明示的な特殊化）**</span>を利用する。

---
## **完全特殊化の方法**

```c++
// T 型の値を合成する関数テンプレート宣言、定義
template<class T>
void combine(T l, T r)
{
   T res = l + r;
   std::cout << res  << std::endl;
}


// T を const char*型として扱う場合の関数を宣言、定義
// 完全特殊化 
template<>
void combine<const char*>(const char* l, const char* r)
{
    std::cout << l << r << std::endl;
}

```
下側の ```combine``` 関数に注目する。  
```template<class T>``` から ```class T``` がなくなり、  
代わりに ```combine``` 関数にテンプレート実引数が指定され、  
関数テンプレートで仮引数だった所が、全て具体的な型になっている。

定義の内容を見ると **加算** はしておらず、  
二つの引数を繋げて出力する様になっている。

こうする事で、引数が ```const char*``` つまり文字列の場合のみ、  
具体的な型で定義した関数が処理され、文字列がつながる形で出力されるようになる。

## **クラステンプレートの完全特殊化**

関数テンプレートだけではなく、クラステンプレートも完全特殊化が可能。

```c++
#include <iostream>
#include <string>

// クラステンプレート
template<class T>
class Data {

public:
    // コンストラクタ
    Data(T l, T r)
    :valueL_(l), valueR_(r){}
    // デストラクタ
    virtual ~Data() {}

public:
    // データを表示するメンバ関数の宣言
    void disp() const
    {
        // 乗算する
        std::cout << valueL_ * valueR_ << std::endl;
    }

private:
    T valueL_ = {}; // データ
    T valueR_ = {}; // データ
};


// T が std::string の場合の完全特殊化
template<>
class Data<std::string> {
public:
    // コンストラクタ
    Data(std::string l, std::string r)
        :valueL_(l), valueR_(r) {}
    // デストラクタ
    virtual ~Data() {}

public:
    // データを表示するメンバ関数の宣言
    void disp() const
    {
        // 続けて文字を表示する
        std::cout << valueL_ << valueR_ << std::endl;
    }

private:
    std::string valueL_ = {}; // データ
    std::string valueR_ = {}; // データ
};

// エントリー関数
int main()
{
    Data<int> iData(5, 2);
    iData.disp();

    Data<std::string> stringData("今日は", "雨");
    stringData.disp();

    return 0;
}
```

テンプレート実引数にクラス型を指定するといった事も可能なので、  
用途に合わせて色々試してみてほしい。
