# **課題**

---
## **課題①　可変長テンプレートで合計値を計算する関数を作成してください**

引数個数が可変の関数テンプレートを実装してください。

- `sumAll(...)` を可変長テンプレートで実装する
- 3 個以上の引数を受け取り、合計値を返す
- `int` `double` 混在入力でも動作するようにする
- 呼び出し例を複数用意して結果を表示する

---
## **課題②　可変長引数を使ったログ出力関数を作成してください**

可変長テンプレートを使って柔軟なログ関数を作成してください。

- `printLog(...)` を実装し、任意個の引数を順に表示する
- 各引数の区切り文字を指定できるようにする
- 引数なし呼び出しも安全に処理する
- 再帰展開または fold 式のどちらかで実装する

---
## **課題③　constexpr if を使って型ごとに分岐する関数を作成してください**

テンプレート引数の型に応じて処理を切り替える関数を実装してください。

- `printTypeInfo(T value)` を作成する
- 整数型・浮動小数点型・文字列型で処理を分岐する
- 分岐には `if constexpr` と型特性を使う
- 実行結果を表示して分岐が正しく働くことを確認する

---
## **課題④　concept で制約を付けたテンプレート関数を作成してください**

不正な型引数をコンパイル時に防ぐ関数を実装してください。

- 算術型のみ受け付ける `average` 関数テンプレートを作成する
- 制約は `concept` を定義して付与する
- 有効な呼び出し例と無効な呼び出し例をコメントで示す
- 制約なし版と比較して、エラーメッセージの分かりやすさを確認する
