# STL(Standard Template Library)

---
## **テンプレートを活用したC++の標準ライブラリ**

前の授業で紹介したテンプレートを活用した便利な機能が、  
C++で標準ライブラリとして提供されており、  
用途に応じて利用する事ができる。

実際の業務でもよく利用される為、  
こういったものがある、という事はぜひ覚えておいてほしい。

主に以下の内容を取り扱える。  

 - コンテナ
 - イテレータ
 - 関数オブジェクト
 - アルゴリズム

---
### **コンテナ**
**コンテナ** とは<span style="color: red;">**オブジェクト集まりを扱うデータ構造**</span>の総称。  
複数のオブジェクトを管理する際に利用される。  
<span style="color: red;">**コレクション**</span>とも呼ばれる。

下記は stl のコンテナの一例になる。

- array
- vector
- map
- list

また、以前の授業の中で挙げた  **stack** や **queue** も、  
コンテナを内側で持っている **コンテナアダプタ** という分類で用意されている。

非常に便利な半面、一部のコンテナには要素の追加時に、  
**全て作り直されて新しく内容をコピーする処理が行われる**といったデメリットもある。  
それぞれの特徴を把握した上で利用する必要がある。

---
### **イテレータ**
**イテレータ** とは<span style="color: red;">**コンテナが持つオブジェクトに順番にアクセスする為のオブジェクト**</span>。  
<span style="color: red;">**反復子**</span> とも呼ばれる。

各コンテナには、コンテナの型に対応したイテレータが用意されており、  
コンテナに登録されているオブジェクトを順に処理しやすいようになっている。  
ポインタの様に扱う事で利用できる。

抽象的に反復処理を行うための物である為、  
どの型のイテレータであっても、基本的な使い方は同じになる。


---
### **関数オブジェクト**
**関数オブジェクト** とは <span style="color: red;">**関数のように利用できるオブジェクト**</span>の総称。  
stl には、四則演算や比較演算を行う関数オブジェクトが用意されている。

具体的には **()演算子オーバーロード** を用意したクラスが実装されている形になっており、  
自身で実装する事も容易になっている。

 **関数オブジェクト** という言葉自体は、  
言語問わず使われており、C++ 独自の物ではない。

---
### **アルゴリズム**
**アルゴリズム** は、その名の通り、  
コンテナの並べ替えや検索、コピーや入れ替えを行う。  

アルゴリズムはコンテナから独立しており、  
アルゴリズム（関数）にコンテナを渡す事で利用する。

一部のコンテナはメンバ関数としてアルゴリズム関数を持っている。  
その場合は、メンバ関数側のアルゴリズム関数を利用した方が好ましい。
