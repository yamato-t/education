# **アルゴリズム**

---
## **stl のアルゴリズム**
検索やソート等、stl では多数のアルゴリズム関数を提供している。  
引数や戻り値には基本的にイテレータを利用する。  

---
## **検索を行う find**

下記は、vector から特定の値を検索する例になる。

```c++
#include <iostream>

// find を扱う
#include <algorithm>
#include <vector>

// エントリー関数
int main()
{
	// int を扱う vector
	std::vector<int> src = {10, 2, 4, 5, 9, 7};

	// src から「９」を探し、最初に見つかった位置（イテレータ）を返す
	std::vector<int>::iterator it = std::find(src.begin(), src.end(), 9);

	if (it == src.end()) 
	{
		std::cout << "見つからなかった" << std::endl;
	}
	else 
	{
		std::cout << "見つかった" << std::endl;
	}
}

```
```std::find``` で、検索したいコンテナの範囲(```begin``` から ```end```)を渡し、  
最後に検索する値を渡している。

検索する値が最初に見つかった位置を返し、
見つからなかった場合は末端の位置を返す。  

---
## **条件付きで検索を行う find_if**

下記は、vector から特定の値を条件付きで検索する例になる。

```c++
#include <iostream>

// find_if を扱う
#include <algorithm>
#include <vector>

// チェック関数
bool check(int value)
{
	// １を足すと１０になる値なら true を返す
	return (value + 1) == 10;
}

// エントリー関数
int main()
{
	// int を扱う vector
	std::vector<int> src = {10, 2, 4, 5, 9, 7};

	// src から check 関数の結果が true になる値を探し、その位置（イテレータ）を返す
	std::vector<int>::iterator it = std::find_if(src.begin(), src.end(), check);


	if (it == src.end())
	{
		std::cout << "見つからなかった" << std::endl;
	}
	else
	{
		std::cout << "見つかった" << std::endl;
	}
}
```
```find``` と違う点は、最後の引数が関数になっている事。  
```check``` 関数の引数にコンテナの内容が一つずつ入り、  
条件チェックされ、戻り値が true になった位置を返す。

どんな関数でも渡せるわけではなく、  
コンテナと ```find_if``` 関数が対応している関数にする必要がある。  
具体的には下記が挙げられる。

- 戻り値はbool型
- 引数の型がコンテナの実引数と同じ
- 引数の数が一つだけ

## **並べ替えを行う sort**

下記は、vector を並べ替える例になる。

```c++
#include <iostream>

// sort を扱う
#include <algorithm>
#include <vector>

// エントリー関数
int main()
{
	// int を扱う vector
	std::vector<int> src = {10, 4, 5, 2, 9, 7};

	// src の内容を並べ替える
	std::sort(src.begin(), src.end());

	// 昇順に表示される
	for (std::vector<int>::iterator it = src.begin(); it != src.end(); ++it)
	{
		std::cout << *it << std::endl;
	}
}
```
```std::sort``` で、並べ替えたいコンテナの範囲(```begin``` から ```end```)を渡している。  
これによって内容が全て昇順に並び変えられる。

---
### **並べ替える順番を変えたい場合**

上記は昇順に並び変えられるが、  
降順にしたい場合も出てくる。

そのような時は、並び替えの条件となる関数を引数に渡す事で、  
並び替えの方法を変更する事ができる。

```c++
#include <iostream>

// sort を扱う
#include <algorithm>
#include <vector>

// 二つの値を比較して第一引数の方が大きい場合に true を返す関数
bool greater(int l, int r)
{
	return l > r;
}


// エントリー関数
int main()
{
	// int を扱う vector
	std::vector<int> src = {10, 4, 5, 2, 9, 7};

	// 最後の引数に比較する関数を渡す
	std::sort(src.begin(), src.end(), greater);

	// 降順に表示される
	for (std::vector<int>::iterator it = src.begin(); it != src.end(); ++it)
	{
		std::cout << *it << std::endl;
	}
}
```
```sort``` 関数の最後の引数が関数になっている。  
```greater``` 関数の引数にコンテナ内の二つの値が引数として渡され、値の比較が行われる。  
その戻り値によって、並び替えられる順番が変わる。  

第一引数の方が大きい場合に true を返すと降順に、  
逆に false を返すと昇順になる。

なお、引数に先ほど紹介した ```std::greater``` を渡す事も可能。  
こちらも同じ様に降順に並べ替えられる
```c++
// 第一引数が大きい場合に true を返す関数オブジェクト
std::greater<int> func;
// src の引数に渡す
std::sort(src.begin(), src.end(), func);
```

find_if と同じくどんな関数でも渡せるわけではない。  
コンテナと ```sort``` 関数に対応する関数にする必要がある。
