## **課題①　関数オーバーロードで複数型の計算を行うプログラムを作成してください**

同名関数を型ごとに使い分けるプログラムを作成してください。

- `add` 関数を `int` `float` `double` 用にオーバーロードする
- `add(int, int, int)` の 3 引数版も追加する
- 各オーバーロードが呼ばれたことを結果表示で確認できるようにする
- 戻り値と表示結果の型差を確認する

---
## **課題②　演算子オーバーロードで2次元ベクトル計算を実装してください**

`Vector2` クラスで演算子を定義するプログラムを作成してください。

- `x` `y` を持つ `Vector2` クラスを作成する
- `+` `-` `==` をオーバーロードする
- `std::cout` で表示できるよう `<<` もオーバーロードする
- 複数の計算結果をコンソールへ表示して確認する
