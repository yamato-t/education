# **スレッドの実装例**

---

## **std::thread**

C++ でスレッドを扱う為には `<thread>` ヘッダを使う。  
C++11 以降に追加された <span style ="color: red;">**std::thread**</span> を利用することで、  
標準ライブラリだけでスレッドを作成・管理できる。

---

## **基本的な使い方**

最もシンプルなスレッドの作成例を見てみる。

```c++
#include <iostream>
#include <thread>

// スレッドで実行する関数
void Task()
{
    std::cout << "別スレッドで実行中" << std::endl;
}

int main()
{
    // スレッドを作成して Task 関数を実行させる
    std::thread t(Task);

    // スレッドの終了を待つ
    t.join();

    std::cout << "メインスレッド終了" << std::endl;
    return 0;
}
```

`std::thread` のコンストラクタに実行したい関数を渡すと、  
その時点でスレッドが生成されて関数の実行が始まる。

<span style ="color: red;">**join()**</span> は、そのスレッドの処理が終わるまで  
呼び出し元のスレッドを待機させる関数となる。  
`join()` を呼ばずにスレッドオブジェクトが破棄されると  
プログラムが強制終了してしまう為、必ず呼ぶ必要がある。

---

## **引数を渡す**

スレッドに渡す関数に引数がある場合、  
コンストラクタの第二引数以降に値を渡す。

```c++
#include <iostream>
#include <thread>
#include <string>

// 引数付きの関数
void Count(int start, int end, std::string_view name)
{
    for (int i = start; i <= end; ++i)
    {
        std::cout << name << ": " << i << " ";
    }
    std::cout << std::endl;
}

int main()
{
    // スレッドA: 1 〜 5 をカウント
    std::thread tA(Count, 1, 5, "tA");

    // スレッドB: 6 〜 10 をカウント
    std::thread tB(Count, 6, 10, "tB");

    // 両スレッドの終了を待つ
    tA.join();
    tB.join();

    return 0;
}
```

出力はスレッドの実行順序によって変わる場合が殆ど。  
二つのスレッドがどちらが先に実行されるかは、  
OS のスケジューリングに依存する為、毎回異なる可能性が高い。

---

## **ラムダ式との組み合わせ**

`std::thread` はラムダ式と組み合わせて使う事も多い。

```c++
#include <iostream>
#include <thread>

int main()
{
    int value = 100;

    // ラムダ式を使ってスレッドを作成
    std::thread t([value]()
    {
        std::cout << "値は " << value << std::endl;
    });

    t.join();

    return 0;
}
```

関数を別途定義しなくて済む為、  
処理が短い場合や一度しか使わない場合に便利。

---

## **detach について**

`join()` の代わりに <span style ="color: red;">**detach()**</span> を呼ぶ方法もある。  
`detach()` を呼ぶと、スレッドをメインスレッドから切り離し、  
独立して動かすことができる。

```c++
std::thread t(Task);
t.detach(); // メインスレッドと切り離して独立して動かす
```

ただし `detach()` したスレッドは完了を待機できない。  
メインスレッドが先に終了するとプロセスごと終わってしまい、  
途中のスレッドも強制終了される為、扱いには注意が必要となる。
