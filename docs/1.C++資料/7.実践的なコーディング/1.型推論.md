# **型推論**

---

## **型の指定**

変数を宣言する際に、型を指定して宣言する必要がある事はわかると思う。

```c++
// int 型の変数を宣言する
int value = 100;
```

ところが昨今の言語には、 **記述内容から型を推論する** という便利な仕様がある。  
これを **型推論** と呼び、C++ でも一時変数等で利用する事ができる。

```c++
// 代入される値から int 型になる
auto value1 = 100;

// 代入される値から float 型になる
auto value2 = 0.1f;

// 代入される値から value1（つまり int 型）のポインタ型になる
auto value1Ptr = &value1;
```

ただし、参照の型推論だけは少し工夫がいる。  
実は参照は、```auto``` のみで扱う事が出来ない。  

```c++
// int 型の変数とその参照を宣言する
int  value = 0;
int& ref   = value;

// temp1 は int 型になってしまう
auto temp1 = ref;
```

方法としては二つある。  
一つは ```auto``` の後ろに & を付ける方法。  
もう一つは ```decltype``` というキーワードを利用する方法。  

```c++
// temp1 も temp2 も int& 型になる
auto&          temp1 = ref;
decltype(auto) temp2 = ref;
```

```decltype``` は「 **型を取得する** 」という働きをする。  
テンプレートを上手に使った関数を自作する際等、  
利用する機会が増えると思うので覚えておいてほしい。


そして型推論は、**関数の戻り値** にも利用できる。
```c++
// 戻り値が int 型になる事が推論出来る
auto func()
{
    // int 型の型推論
    auto i = 100;
    return i;
}
```

ただし参照を返したい場合は、変数の時と同じ問題が発生する。

```c++
// int 型の変数とその参照をグローバル宣言する
int   value = 0;
int&  ref   = value;

// int& 型を戻したいが int 型になってしまう
auto func()
{
    // 参照を返す
    return ref;
}
```

改善する為には、同じように ```decltype``` を利用する。

```c++
// auto を decltype で囲み、auto の型を取得して関数の戻り値型とする
// これで戻り値型が int& になる
decltype(auto) func()
{
    // 参照を返す
    return ref;
}
```

なお関数の引数は ```auto``` に出来ない。コンパイルエラーになる。

##  **型推論と関数の補足**

型推論と関数の関係を、もう少し補足する。  
関数を作成する際に「 戻り値型 関数名 (引数){} 」という順番で作成していたと思う。  
ただ C++ バージョン11 以降では、上記以外の表記方法もとれる。

まずこういった形の関数があるとする。

```c++
// 引数を二倍にして返す関数
int func(int val)
{
    auto ret = val * 2;
    return ret;
}
```

ここから、戻り値の型指定を後ろ側に移動させる事が出来る。

```c++
// 引数を二倍にして返す関数
auto func(int val) -> int
```

まず最初の型指定部分が具体的な型ではなく、auto になっている。  
そして、引数の後に矢印と具体的な型を指定する。

ここではまだ具体的に int 型を指定しているが、  
実は int のキーワードを使わずに、戻り値を「引数と同じ型」に指定する事が出来る。

```c++
// 引数を二倍にして返す関数
auto func(int val) -> decltype(val)
```

```decltype``` は「型を取得する」という働きをする為、  
引数を ```decltype``` で囲めば int と記述した事と同じになる。

さらに C++ バージョン 14 以降では -> 以降を省略する事が出来る。  
その結果、型推論の中で紹介したような形に落ち着く。

```c++
// 引数を二倍にして返す関数
auto func(int val)
{
    auto ret = val * 2;
    return ret;
}
```
