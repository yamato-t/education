# **演算子オーバーロード**

---

コーディングには色々な記号が利用できるが、  
これらが**演算子**と呼ばれる事は知っていると思う。  
演算子は役割に応じて名前が付けられている。

|  名称  |  例  |
| ---- | ---- |
|  算術演算子  | +, -, * |
|  比較演算子  | ==, !=, <, > |
|  論理演算子  | !, &&, ││, |

…等々

<span style="color: red;">**これらの演算子は関数オーバーロードと同じ様にオーバーロード可能**</span> 。  
**そのクラスにおいて特殊な意味を持つ演算子**として再定義する事が出来る。  

---

## **演算子オーバーロードの宣言と定義**

演算子オーバーロードの宣言と定義も関数と同じように考えて貰えればいいが、  
記述が特殊で **operator** というキーワードを利用する必要がある。

```c++
// 動物クラス
class Animal final
{
public:
	// +=演算子のオーバーロード(足の数を追加する)
	void operator +=(int addFoot);

	// +演算子のオーバーロード(足の数を合計した結果の数を返す)
	int operator +(const Animal& animal);

public:
	std::string		name_		= {};
	int				footNum_	= 0;
};

// += 演算子オーバーロード(足の数を追加する)
void Animal::operator +=(int addFoot)
{
	// 自身の足の数を足すだけ
	footNum_ += addFoot;
}

// + 演算子オーバーロード(足の数を合計した結果の数を返す)
int Animal::operator +(const Animal& animal)
{
	// 自身の足の数と、引数で渡されたインスタンス（参照）の足の数を合計し、戻り値として返す
	return footNum_ + animal.footNum_;
}

```

上記は、
```c++
+=
+
```
の二つの算術演算子をオーバーロードして、  
Animal クラスインスタンス用の特殊な挙動を行うようにしている。

---
## **演算子オーバーロードの利用方法**

演算子オーバーロードを定義している事によって、  
下記の様にクラスのインスタンスに対して算術演算子を利用する事が出来る様になる、

```c++
// インスタンスを作成する
Animal animal1("動物１", 2);
Animal animal2("動物２", 4);

// animal1 の足の数に２を加える
// 本来はクラスのインスタンスに「+=」の演算子は利用できない
// だが、演算子オーバーロードを宣言、定義した事によって
// 独自に定義した内容の関数が呼び出される形になる
animal1 += 2;

// animal1 と animal2 の足の数を足した結果を返す
// こちらも本来はインスタンス同士「+」の演算子は利用できない
int total = animal1 + animal2;
std::cout << "足の総数は" << total << "本" << std::endl;
```
