# **コンストラクタ**

---

<span style="color: red;">クラスをインスタンス化する際に呼び出される特別な関数</span>。  
主にメンバ変数の初期化や最初にやっておきたい処理に利用される。  

---

## **コンストラクタの宣言と定義**

コンストラクタは特殊な**関数**なので、  
関数のように宣言、定義する。  

```c++
#include <string>
#include <iostream>

// 動物クラス
class Animal final
{
public:
	// コンストラクタの宣言
	Animal();

public:
	std::string		name_		= {};
	int				footNum_	= 0;
};

// コンストラクタの定義
Animal::Animal()
{
	std::cout << "コンストラクタが呼ばれました" << std::endl;
	name_ = "動物";
	footNum_ = 4;
}

```

コンストラクタの名前は**クラス名と同一**になる。  
また**戻り値はない**。

コンストラクタで利用できる「メンバイニシャライザ」という方法で、  
メンバを初期化する事も出来る。


```c++

// コンストラクタの定義 ( メンバイニシャライザでメンバ変数を初期化 )
Animal::Animal()
:name_("動物")
,footNum_(4)
{
	std::cout << "コンストラクタが呼ばれました" << std::endl;
}

```

## **引数付きコンストラクタの宣言と定義**

コンストラクタは関数である為、引数を受け取る事も可能。  

```c++
// 動物クラス
class Animal final
{
public:
	// 引数付きコンストラクタの宣言
	Animal(const char* name, int footNum);

public:
	std::string		name_		= {};
	int				footNum_	= 0;
};

// 引数付きコンストラクタの定義
Animal::Animal(const char* name, int footNum)
{
	std::cout << "引数付きコンストラクタが呼ばれました" << std::endl;

	name_		= name;
	footNum_	= footNum;
}
```

今までのコンストラクタはコードに内容を直接記述している為、  
インスタンス化した時に同じ初期化が行われるが、  
上記の様に引数で渡す事で、初期化の内容を変える事が出来る様になる。  

---
## **コンストラクタの呼び出し方**

コンストラクタは<span style="color: red;">インスタンス化された際に処理される物</span>である為、  
<span style="color: red;">**変数宣言しただけで呼び出される**</span>。  

```c++
// インスタンス化    
{
	// 変数宣言であり、関数呼び出しには見えないがちゃんと呼び出されている
	// 通常の宣言では通常のコンストラクタが呼ばれる
	Animal animal1;

	// 宣言時の変数の後ろに引数の形で指定すると、引数付きコンストラクタが呼ばれる
	Animal animal2("猛獣", 2);
}
```

なお、意図して呼び出す事も可能。

---
## **コンストラクタは省略可能**

コンストラクタは省略してもコンパイルエラーにならない。  
省略した場合、何もしないコンストラクタが自動で定義される。
