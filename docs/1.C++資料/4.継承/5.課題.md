# **課題**
---

## **課題①　下記の要件を満たすプログラムを作成してください**

細かな実装方法の指定はありません。自由に行ってください。  
ただし**下記の要件は満たしてください**。

- コンソールプロジェクトで作成し、各処理の結果はコンソールに表示してください。
- 動物クラスを定義し「鳴く」という純粋仮想関数を宣言してください。
- 犬クラスと猫クラスを定義し、動物クラスを継承してください。
- 「鳴く」の実装内容も自由ですが、犬クラスと猫クラスで違う内容にしてください。
- 「鳴く」の結果もコンソールに表示するようにしてください。
- メイン関数で、犬クラス、猫クラスをインスタンス化し、多態性を実現した形で鳴かせてください。
