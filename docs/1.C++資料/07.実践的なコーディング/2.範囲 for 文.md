# **範囲 for 文**
---

## **for 文の記述**

ある配列があり、その内容を全てコンソールに出力するとなった場合、  
この様な書き方をしていたと思う。

```c++
// 配列を宣言して初期化する
int v[] = { 0,1,2,3,4,5 };

// for 文で全て走査して出力する
for (int i = 0; i < 6; ++i) {
	std::cout << v[i] << std::endl;
}
```

C++ バージョン11 以降は **範囲 for 文** という記述で、  
型推論と合わせて下記の様に簡潔にする事が出来るようになった。

```c++
// 範囲 for 文で全て走査して出力する
for (auto num : v) {
	std::cout << num << std::endl;
}
```

```v``` の各要素が順番に ```num``` に入るようになる。

ただし、```num``` の型が ```auto(int)``` である為、  
この書き方ではオブジェクトの複製が行われており、内容を変更する事は出来ない。  
変更する為には、```auto&``` の形で参照を取得する必要がある。

```c++
// 範囲 for 文で全て走査して出力する
for (auto& num : v) {
	num = 0; // v の各要素が更新される
}
```

範囲 for 文 は ```std::vector<T>``` といったコンテナでも利用できる。  
イテレータで走査するよりかなり簡潔になる。

```c++
// コンテナを初期化する
std::vector<int> v = {0,1,2,3,4,5};

// 内容を全て出力する
for (auto num : v) {
	std::cout << num << std::endl;
}
```
