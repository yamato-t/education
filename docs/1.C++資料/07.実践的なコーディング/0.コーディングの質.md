# **実用的なコーディング**

---
## **コーディングの質**

ゲーム作品を作っている為、  
「面白い」や「絵的なインパクト」という物が成果物の評価になるが、  
<span style="color: red;">**プログラムの場合、勿論コーディングそのもの評価対象になる**</span>。

- **見る人の事を考えてコメント等を用意しているか**
- **不正なメモリを見てしまうようなコードを避けているか**
- **ちゃんと設計を考えたクラスの実装になっているか**
- **多態性をうまく利用した処理を実装出来ているか**
- **テンプレートなどを必要に応じて活用しているか**

これらを意識し、質の高いコーディングを目指さなければならない。

## **C++ のバージョン**

加えて<span style="color: red;">**昨今の標準的な C++ バージョンを利用しているか**</span>という点も軽視できない。  
というのも言語は C++ に限らず日々進化してく。  
昨今の言語バージョンに合わせた実用的なコーディングを心掛ける必要がある。  

少々古い物もあるが、現場でも頻繁に使われており、  
学生でも取り入れる事が容易な幾つかのコーディングを紹介しておく。  

あくまで言語の進化に合わせて、  
新しく生まれてきた物のごく一部に過ぎないので、  
是非バージョンについて色々と調べてみてほしい。

現行の最新 C++ バージョンは 23 になる。  
ただし改訂予定のバージョンとしてすでに 26 が存在している。
