# **課題**

---
## **課題①　クラスの基本要素を使ってキャラクター管理プログラムを作成してください**

`Character` クラスを定義し、  
クラスの基本要素を確認できるプログラムを作成してください。

- `name` `hp` `attack` をメンバ変数として持たせる
- メンバ変数は `private`、取得・更新は `public` メンバ関数経由にする
- `printStatus()` を作成して現在の状態を表示する
- `main` で 2 体分のインスタンスを作成して表示する

---
## **課題②　コンストラクタ・コピーコンストラクタ・デストラクタを確認するプログラムを作成してください**

インスタンスの生成と破棄の流れを確認するプログラムを作成してください。

- 引数なしコンストラクタと引数付きコンストラクタを実装する
- コピーコンストラクタを実装し、呼び出されたことが分かるログを表示する
- デストラクタを実装し、破棄タイミングをログで確認できるようにする
- 値渡し関数を作成し、コピーコンストラクタが呼ばれる場面を作る

---
## **課題③　静的メンバと const メンバ関数を利用するプログラムを作成してください**

クラス全体で共有する値と、更新しない関数の使い分けを確認してください。

- 生成中インスタンス数を数える `static` メンバ変数を実装する
- コンストラクタとデストラクタでインスタンス数を増減する
- 状態表示関数は `const` メンバ関数として実装する
- `static` メンバ関数で現在のインスタンス数を表示する

---
## **課題④　名前空間で同名クラスを使い分けるプログラムを作成してください**

名前空間の役割を確認するプログラムを作成してください。

- `GameA` と `GameB` の 2 つの名前空間を作る
- 両方の名前空間に同名の `Player` クラスを定義する
- それぞれに異なる `printInfo()` を実装する
- `main` で明示的に名前空間を指定して両方を利用する
