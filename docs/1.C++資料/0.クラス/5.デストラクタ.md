# **デストラクタ**

---

<span style="color: red;">**クラスのインスタンスが破棄されるときに呼び出される特殊な関数**</span>。  
そのインスタンスの後始末処理がある場合に利用される。   

---

## **デストラクタの宣言と定義**

デストラクタも特殊な**関数**なので、  
関数のように宣言、定義する。  

```c++
// 動物クラス
class Animal final
{
public:
	// デストラクタの宣言
	~Animal();

public:
	std::string		name_		= {};
	int				footNum_	= 0;
};

// デストラクタの定義
Animal::~Animal()
{
	std::cout << "デストラクタが呼ばれました" << std::endl;

	name_ = "破棄済み";
	footNum_ = 0;
}
```

デストラクタの名前は**クラス名の前に~が付いた名前**になる。  
こちらも**戻り値はなく**、  
デストラクタの場合は**引数もない**。

---
## **デストラクタの呼び出し方**

デストラクタは<span style="color: red;">**インスタンスが破棄される際に処理される**</span>。  
その為、クラスインスタンスとして宣言した変数の寿命がなくなる時に呼び出される

```c++

// インスタンス化と破棄
{
	Animal animal;

	std::cout << "スコープを抜ける" << std::endl;
}
// スコープを抜けると animal (インスタンス) が破棄され、メモリ上からも消える
// つまりスコープを抜けたタイミングで animal のデストラクタが呼ばれる

```

ちなみに、デストラクタも意図して呼び出す事が可能。

---
## **デストラクタは省略可能**

デストラクタは省略してもコンパイルエラーにならない。  
省略した場合、何もしないデストラクタが自動で定義される。
