# **自分自身のアドレス（this）**

---

クラス内で、自分自身のインスタンスのアドレスを知る事が出来る。  
上記には**this**キーワードを利用する。

---

## **thisの利用方法**

メンバ関数の中で、thisキーワードを記述するだけでいい。  
thisは「そのインスタンスのアドレスが入っているポインタ」である為、  
アロー演算子でメンバ変数やメンバ関数を利用する事が出来る。

```c++
// クラス「MyClass」の定義
class MyClass
{
public:
	void functionA(int valueA); // メンバ変数と同じ名前の引数があるメンバ関数

private:
	int valueA = 0;
};

// メンバ変数と同じ名前の引数があるメンバ関数の定義
void MyClass::functionA(int valueA)
{
	// 引数名とメンバ変数名が一致する為、thisキーワードを使って区別する
	this->valueA = valueA;
}
```


---

## **thisの用途**

上記の様な場合に加えて、  
クラス型のポインタを引数に渡す関数を呼び出す場合などにも利用される。

```c++

// MyClass型のポインタが引数になるグローバル関数の定義
void functionC(MyClass* myclassPtr)
{
	myclassPtr->functionA(0);
}

```
この様なグローバル関数があり、  
それをクラスのメンバ関数から呼び出したい場合、  
メンバ関数側では下記の様に記述する。


```c++

// グローバル関数を呼び出すメンバ関数の定義
void MyClass::functionB()
{
	// 引数に自身のインスタンスのアドレスを渡す
	functionC(this);
}

```
