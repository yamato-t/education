# **クラステンプレート**

---
## **同じ機能で扱う型のみが違うクラス**

**スタックやキュー** などのデータ構造では、  
データを追加し、状況に応じて取り出すといった事ができたが、  
**扱う型が違うだけで追加や取り出しの挙動そのものは一緒** 。

int 型のスタック、float 型のスタック等、それぞれ作成するのは手間になる。  
そんな場合に<span style="color: red;">**クラステンプレート**</span>が利用できる。

---
## **クラステンプレートの宣言と定義**

記述や利用するキーワードは関数テンプレートとほぼ同じ。  
特殊化や実体化といった言葉も関数テンプレートの場合と同じ意味を指す。

```c++
#include <iostream>
#include <string>

template<class T>
class Data {

public:
    // コンストラクタ
    Data()  {}
    // デストラクタ
    virtual ~Data() {}

public:
    // データを表示するメンバ関数の宣言
    void disp() const;

    // データを取得するメンバ関数の宣言
    const T& get() const;

    // データを設定するメンバ関数の宣言
    void set(const T& value);

private:
    T value_ = {}; // データ
};


// データを表示するメンバ関数の定義
template<class T>
void Data<T>::disp() const
{
    std::cout << value_ << std::endl;
}

// データを取得するメンバ関数の定義
template<class T>
const T& Data<T>::get() const
{
    return value_;
}

// データを設定するメンバ関数の定義
template<class T>
void Data<T>::set(const T& value)
{
    value_ = value;
}
```

```class Data``` の前に ```template<class T>``` という記述が追加されており、  
メンバ変数やメンバ関数で扱う型が ```T``` になっている。  
これらは関数テンプレートと同じように、テンプレート仮引数を利用する形になっている。

メンバ関数の定義を、宣言と別に行う場合だけ少し違う。  
クラステンプレートのメンバ関数である事を示す為、  
各関数の前に ```template<class T>``` が付き、  
さらにクラス名に ```<T>``` が付いている。  


---
## **クラステンプレートの利用**


```c++
// エントリー関数
int main()
{
    // int 型のデータを扱うクラスのインスタンス
    Data<int> intData;
    intData.set(100);
    intData.disp();

    // string 型のデータを扱うクラスのインスタンス
    Data<std::string> stringData;
    stringData.set("文字列");
    stringData.disp();

    return 0;
}
}
```

利用する側を見ると、 ```<int>```、 ```<std::string>``` という記述がついている。  
これによって、クラステンプレートの ```T``` 型を具体的な型に置き換えたクラスをインスタンス化できる。  
C++のバージョンによっては、コンストラクタ引数から推論される事で、  
**テンプレート実引数を省略する事もできる**。

 
---
## **クラステンプレートの注意点**
クラステンプレートも、関数テンプレートと同じ事が言える。  
その為、クラステンプレートの宣言と定義を別ファイルに記述するとリンクエラーが起こる。

template.h
```c++
#pragma once 

template<class T>
class Data {

public:
    // コンストラクタ
    Data() {}
    // デストラクタ
    ~Data() {}

public:
    // データを表示するメンバ関数の宣言
    void disp() const;

    // データを取得するメンバ関数の宣言
    const T& get() const;

    // データを設定するメンバ関数の宣言
    void set(const T& value);

private:
    T value_ = {}; // データ
};
```


template.cpp
```c++
#include <iostream>
#include "template.h"

// データを表示するメンバ関数の定義
template<class T>
void Data<T>::disp() const
{
    std::cout << value_ << std::endl;
}

// データを取得するメンバ関数の定義
template<class T>
const T& Data<T>::get() const
{
    return value_;
}

// データを設定するメンバ関数の定義
template<class T>
void Data<T>::set(const T& value)
{
    value_ = value;
}

// int 型で利用する事を予め知らせておく
template class Data<int>;

```

template.cpp の最後にあるように、明示的な実体化が必要になる。
