# **ラスタライザステート**

---

「[0.レンダリングパイプライン](../0.レンダリングパイプライン/0.レンダリングパイプライン.md)」で触れた通り、  
ラスタライザは 3D の三角形を 2D ピクセルへ変換する段階を担う。  

「[0.ポリゴン描画](../2.ポリゴン描画/0.ポリゴン描画.md)」では PSO へのセット方法と  
各パラメータの意味をコメントで示した。  
ここでは設定の意図をより掘り下げて説明する。

## **フィルモード**

フィルモードは三角形の内部をどのように描くかを制御する。  

| 設定値 | 説明 |
|--------|------|
| `D3D12_FILL_MODE_SOLID` | 三角形の内部を塗りつぶす（通常の描画） |
| `D3D12_FILL_MODE_WIREFRAME` | 三角形の輪郭（辺）だけを描く |

ワイヤーフレームモードはポリゴン構造の確認やデバッグに用いる。  

```c++
D3D12_RASTERIZER_DESC rastDesc = {};

// 通常の塗りつぶし描画
rastDesc.FillMode = D3D12_FILL_MODE_SOLID;

// ワイヤーフレーム（デバッグ・構造確認用）
rastDesc.FillMode = D3D12_FILL_MODE_WIREFRAME;
```

## **カリング**

<span style ="color: red;">**カリング**</span>とは、描画不要なポリゴンをシェーダ実行前にスキップする仕組みを指す。  

3D のポリゴンには「表面」と「裏面」があり、  
通常カメラから見えない裏面を描く必要はない。  
カリングはこの「不要な面」を自動的に除外し、GPU の処理を削減する。  

### **頂点の巻き順（表裏の判定方法）**

DirectX では、頂点が <span style ="color: red;">**時計回りに並んでいる面**</span> を「表面」と判定する。  
`FrontCounterClockwise = FALSE` でこの既定動作を有効にする。  

裏面（反時計回りに見える面）はカリングでスキップされる。  

### **カリングモードの設定**

| 設定値 | 挙動 | 用途 |
|--------|------|------|
| `D3D12_CULL_MODE_BACK` | 裏面をスキップ | 通常の不透明 3D オブジェクト |
| `D3D12_CULL_MODE_NONE` | カリングなし（両面描画） | 草・布など薄いオブジェクト |
| `D3D12_CULL_MODE_FRONT` | 表面をスキップ | シャドウボリューム等の特殊用途 |

```c++
// 背面カリング（通常はこれを使用）
rastDesc.CullMode              = D3D12_CULL_MODE_BACK;
rastDesc.FrontCounterClockwise = FALSE; // 時計回りを表面とする（DirectX 既定）

// 両面描画（裏から見ても描画される）
rastDesc.CullMode = D3D12_CULL_MODE_NONE;
```

## **ラスタライザステートの設定例**

PSO に設定する標準的なラスタライザステートの設定例。  

```c++
D3D12_RASTERIZER_DESC rastDesc = {};
rastDesc.FillMode              = D3D12_FILL_MODE_SOLID;
rastDesc.CullMode              = D3D12_CULL_MODE_BACK;
rastDesc.FrontCounterClockwise = FALSE;
rastDesc.DepthBias             = 0;       // 深度値のオフセット（通常は 0）
rastDesc.DepthBiasClamp        = 0.0f;
rastDesc.SlopeScaledDepthBias  = 0.0f;
rastDesc.DepthClipEnable       = TRUE;    // 視野範囲外をクリップする
rastDesc.MultisampleEnable     = FALSE;
rastDesc.AntialiasedLineEnable = FALSE;

// PSOに設定
psoDesc.RasterizerState = rastDesc;
```
