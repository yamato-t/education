# **ブレンドステート**

---

実際のゲームでは半透明描画など、様々なブレンド表現が行われる。  
これまでのポリゴン描画は全て不透明で描画されている。  
色の合成方法（ブレンド）を変える事で、半透明やエフェクト描画が可能になる。

## **ブレンドステートとは**

ブレンドステートは描画する色と既に描画されている色を、  
どのように合成するかを定義する。  
半透明表現やエフェクト描画に不可欠な機能。  

ブレンディングの基本式は以下の通り。  

$FinalColor = SourceColor \times SourceBlend \pm DestColor \times DestBlend$

- <span style ="color: red;">**SourceColor**</span>: 今から描画するピクセルの色  
- <span style ="color: red;">**DestColor**</span>: 既にレンダーターゲットに書き込まれている色  
- **SourceBlend** / **DestBlend**: 各色に掛ける係数（設定値によって変わる）

### **ブレンド無効（不透明描画）**

最も基本的な設定。`BlendEnable = FALSE` にするとブレンドは行われず、  
SourceColor がそのままレンダーターゲットに書き込まれる。  

```c++
// ブレンド無効（不透明描画）
D3D12_RENDER_TARGET_BLEND_DESC opaqueDesc = {};
opaqueDesc.BlendEnable           = FALSE;
opaqueDesc.LogicOpEnable         = FALSE;
opaqueDesc.RenderTargetWriteMask = D3D12_COLOR_WRITE_ENABLE_ALL;

D3D12_BLEND_DESC blendDesc = {};
blendDesc.RenderTarget[0] = opaqueDesc;

// PSOに設定
psoDesc.BlendState = blendDesc;
```

### **アルファブレンド（半透明合成）**

最も一般的なブレンドモード。アルファ値に基づいた半透明表現を実現する。  
ガラスや水、パーティクルエフェクトなどの表現に使用される。  

```c++
// アルファブレンド設定
D3D12_RENDER_TARGET_BLEND_DESC alphaBlendDesc = {};
alphaBlendDesc.BlendEnable           = TRUE;
alphaBlendDesc.SrcBlend              = D3D12_BLEND_SRC_ALPHA;
alphaBlendDesc.DestBlend             = D3D12_BLEND_INV_SRC_ALPHA; // INV = Inverse（1 - アルファ値）
alphaBlendDesc.BlendOp               = D3D12_BLEND_OP_ADD;
alphaBlendDesc.SrcBlendAlpha         = D3D12_BLEND_ONE;
alphaBlendDesc.DestBlendAlpha        = D3D12_BLEND_ZERO;
alphaBlendDesc.BlendOpAlpha          = D3D12_BLEND_OP_ADD;
alphaBlendDesc.RenderTargetWriteMask = D3D12_COLOR_WRITE_ENABLE_ALL;

D3D12_BLEND_DESC blendDesc = {};
blendDesc.RenderTarget[0] = alphaBlendDesc;

// PSOに設定
psoDesc.BlendState = blendDesc;
```

この設定により以下の計算が行われる。  

$FinalColor.rgb = SourceColor.rgb \times SourceAlpha + DestColor.rgb \times (1 - SourceAlpha)$

### **加算合成**

ソースカラーとデスティネーションカラーを加算する。  
光や炎、爆発などの発光表現に適しており、  
重ねるほど明るくなる特性を持つ。  

```c++
// 加算合成設定
D3D12_RENDER_TARGET_BLEND_DESC additiveBlendDesc = {};
additiveBlendDesc.BlendEnable           = TRUE;
additiveBlendDesc.SrcBlend              = D3D12_BLEND_SRC_ALPHA;
additiveBlendDesc.DestBlend             = D3D12_BLEND_ONE;
additiveBlendDesc.BlendOp               = D3D12_BLEND_OP_ADD;
additiveBlendDesc.SrcBlendAlpha         = D3D12_BLEND_ONE;
additiveBlendDesc.DestBlendAlpha        = D3D12_BLEND_ZERO;
additiveBlendDesc.BlendOpAlpha          = D3D12_BLEND_OP_ADD;
additiveBlendDesc.RenderTargetWriteMask = D3D12_COLOR_WRITE_ENABLE_ALL;
```

計算式は以下の通り。  

$FinalColor.rgb = SourceColor.rgb \times SourceAlpha + DestColor.rgb$

### **乗算合成**

ソースとデスティネーションを乗算する。  
影や暗闇の表現、色の調整に使用される。  
重ねるほど暗くなる特性を持つ。  

```c++
// 乗算合成設定
D3D12_RENDER_TARGET_BLEND_DESC multiplyBlendDesc = {};
multiplyBlendDesc.BlendEnable           = TRUE;
multiplyBlendDesc.SrcBlend              = D3D12_BLEND_DEST_COLOR;
multiplyBlendDesc.DestBlend             = D3D12_BLEND_ZERO;
multiplyBlendDesc.BlendOp               = D3D12_BLEND_OP_ADD;
multiplyBlendDesc.SrcBlendAlpha         = D3D12_BLEND_ONE;
multiplyBlendDesc.DestBlendAlpha        = D3D12_BLEND_ZERO;
multiplyBlendDesc.BlendOpAlpha          = D3D12_BLEND_OP_ADD;
multiplyBlendDesc.RenderTargetWriteMask = D3D12_COLOR_WRITE_ENABLE_ALL;
```

計算式は以下の通り。  

$FinalColor.rgb = SourceColor.rgb \times DestColor.rgb$

### **減算合成**

デスティネーションからソースを減算する。  
特殊なエフェクトや色の反転表現に使用される。  

```c++
// 減算合成設定
D3D12_RENDER_TARGET_BLEND_DESC subtractBlendDesc = {};
subtractBlendDesc.BlendEnable           = TRUE;
subtractBlendDesc.SrcBlend              = D3D12_BLEND_SRC_ALPHA;
subtractBlendDesc.DestBlend             = D3D12_BLEND_ONE;
subtractBlendDesc.BlendOp               = D3D12_BLEND_OP_REV_SUBTRACT;
subtractBlendDesc.SrcBlendAlpha         = D3D12_BLEND_ONE;
subtractBlendDesc.DestBlendAlpha        = D3D12_BLEND_ZERO;
subtractBlendDesc.BlendOpAlpha          = D3D12_BLEND_OP_ADD;
subtractBlendDesc.RenderTargetWriteMask = D3D12_COLOR_WRITE_ENABLE_ALL;
```

計算式は以下の通り。  

$FinalColor.rgb = DestColor.rgb - SourceColor.rgb \times SourceAlpha$

### **ブレンドモード比較表**

| ブレンドモード | SrcBlend | DestBlend | BlendOp | 用途 |
|----------------|----------|-----------|---------|------|
| 無効（不透明） | - | - | - | 通常の不透明描画 |
| アルファブレンド | SRC_ALPHA | INV_SRC_ALPHA | ADD | 半透明、UI |
| 加算 | SRC_ALPHA | ONE | ADD | 光、炎、爆発 |
| 乗算 | DEST_COLOR | ZERO | ADD | 影、暗闇 |
| 減算 | SRC_ALPHA | ONE | REV_SUBTRACT | 特殊エフェクト |

## **半透明オブジェクトの深度設定**

半透明オブジェクトを描画する際は、  
深度テストとブレンドステートの組み合わせに注意が必要。  

```c++
// 半透明オブジェクト用のデプス設定
D3D12_DEPTH_STENCIL_DESC depthStateDesc = {};
depthStateDesc.DepthEnable    = TRUE;
depthStateDesc.DepthWriteMask = D3D12_DEPTH_WRITE_MASK_ZERO; // 書き込み無効
depthStateDesc.DepthFunc      = D3D12_COMPARISON_FUNC_LESS;
depthStateDesc.StencilEnable  = FALSE;

// PSOに設定
psoDesc.DepthStencilState = depthStateDesc;
```

<span style ="color: red;">**半透明オブジェクトを描画する際の注意点**</span>

- 深度書き込みを無効化する（`DepthWriteMask = ZERO`）
- 深度テストは有効にする（`DepthEnable = TRUE`）
- 不透明オブジェクトを先に描画する
- 半透明オブジェクトは奥から手前の順で描画する
- ソート処理による描画順制御が必要となる

半透明と深度の関係はかなり相性が悪い。
